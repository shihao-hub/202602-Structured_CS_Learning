# 计算机核心课程学习项目 (CS Core Courses)

面向408统考的计算机科学核心课程学习项目，使用Go语言实现，覆盖数据结构、操作系统、计算机组成原理和计算机网络四大科目。

## 408考点覆盖

| 科目 | 分值 | 覆盖模块 |
|------|------|---------|
| 数据结构 | 45分 | `data_structures/` |
| 计算机组成原理 | 45分 | `computer_architecture/` |
| 操作系统 | 35分 | `operating_system/` |
| 计算机网络 | 25分 | `computer_networks/` |

## 项目结构

```
CS_Core_Courses/
├── data_structures/              # 数据结构与算法
│   ├── basic/                   # 基础：数组、链表、栈、队列、哈希表
│   ├── linear/                  # 进阶：串、稀疏矩阵
│   ├── tree/                    # 树：二叉搜索树、堆
│   ├── graph/                   # 图：遍历、最短路径、拓扑排序
│   └── algorithm/               # 算法：排序、查找、DP、贪心、回溯、KMP
├── operating_system/             # 操作系统
│   ├── process/                 # 进程管理与调度算法
│   ├── memory/                  # 内存管理（分页、分段、段页式）
│   ├── synchronization/         # 进程同步（信号量、互斥锁）
│   ├── filesystem/              # 文件系统（inode、目录、分配方式）
│   └── scheduling/              # 磁盘调度与死锁处理
├── computer_architecture/        # 计算机组成原理
│   ├── cpu/                     # CPU：寄存器、ALU
│   ├── memory/                  # 存储器：Cache、虚拟内存
│   ├── instruction_set/         # 指令系统：寻址方式
│   ├── pipeline/                # 流水线：冲突检测
│   └── bus/                     # 总线（理论文档）
├── computer_networks/            # 计算机网络
│   ├── application/             # 应用层：HTTP
│   ├── transport/               # 传输层：TCP
│   ├── network/                 # 网络层：IP、路由、ARP
│   ├── datalink/                # 数据链路层：以太网、差错检测、滑动窗口
│   ├── physical/                # 物理层（理论文档）
│   └── protocols/               # 协议：DNS
├── main.go                      # 主程序入口
└── go.mod                       # Go模块文件
```

## 快速开始

```bash
# 环境要求：Go 1.21+
go version

# 编译验证
cd CS_Core_Courses
go build ./...

# 运行所有示例
go run main.go
```

## 各模块说明

### 数据结构与算法 (45分)
- **基础结构**: 数组、链表、栈、队列、哈希表
- **线性进阶**: 串操作、稀疏矩阵三元组与快速转置
- **树与图**: BST、堆、图遍历、Dijkstra
- **排序算法**: 冒泡、选择、插入、希尔、归并、快速、堆排序、基数排序
- **查找算法**: 顺序、二分、分块查找
- **高级算法**: 动态规划、贪心、回溯、KMP字符串匹配

### 操作系统 (35分)
- **进程管理**: PCB、FCFS/SJF/RR调度
- **内存管理**: 首次/最佳/最差适应、分页、分段、段页式
- **进程同步**: 信号量、互斥锁、生产者消费者
- **文件系统**: inode、目录结构、连续/链接/索引分配
- **调度与死锁**: 磁盘调度(FCFS/SSTF/SCAN/C-SCAN/LOOK)、银行家算法

### 计算机组成原理 (45分)
- **CPU**: 寄存器、ALU运算
- **存储器**: Cache三种映射、LRU/FIFO替换、页面置换算法
- **指令系统**: 指令格式、8种寻址方式
- **流水线**: 五级流水线、数据冲突检测、转发机制
- **总线**: 分类、仲裁、带宽计算（理论）

### 计算机网络 (25分)
- **应用层**: HTTP协议
- **传输层**: TCP三次握手/四次挥手
- **网络层**: IP分组、子网划分、路由算法(RIP/OSPF)、ARP
- **数据链路层**: 以太网、CRC/海明码、滑动窗口(GBN/SR)
- **物理层**: 奈奎斯特/香农定理、编码、复用（理论）
- **协议**: DNS解析
