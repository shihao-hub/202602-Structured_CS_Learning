# 操作系统 (Operating Systems)

本目录包含了操作系统核心概念的Go语言实现，帮助你深入理解计算机操作系统的内部机制。

## 目录结构

### [进程管理](./process/)
- **进程(Process)** - 进程状态管理、PCB结构
- **进程调度(Process Scheduling)** - FCFS、SJF、优先级调度、时间片轮转
- **进程间通信(IPC)** - 管道、消息队列、共享内存
- **进程同步(Process Synchronization)** - 信号量、互斥锁
- **线程(Thread)** - 用户线程 vs 内核线程

### [内存管理](./memory/)
- **内存分配(Memory Allocation)** - 连续分配、非连续分配
- **分页(Paging)** - 页表、TLB、页面置换算法
- **分段(Segmentation)** - 段表、分段机制
- **虚拟内存(Virtual Memory)** - 请求分页、页面置换
- **内存碎片(Memory Fragmentation)** - 内部碎片、外部碎片

### [文件系统](./filesystem/)
- **文件系统结构(File System Structure)** - 目录、inode、超级块
- **文件操作(File Operations)** - 创建、删除、读写
- **目录管理(Directory Management)** - 层次目录结构
- **磁盘调度(Disk Scheduling)** - FCFS、SCAN、C-SCAN
- **文件系统实现(File System Implementation)** - 日志文件系统

### [进程调度](./scheduling/)
- **调度算法(Scheduling Algorithms)** - 先来先服务、短作业优先、优先级调度
- **实时调度(Real-time Scheduling)** - 速率单调调度、最早截止时间优先
- **多级队列调度(Multilevel Queue Scheduling)** - 多级反馈队列
- **调度性能指标(Scheduling Metrics)** - 周转时间、等待时间、响应时间

### [进程同步](./synchronization/)
- **同步问题(Synchronization Problems)** - 生产者消费者、读者写者、哲学家进餐
- **死锁(Deadlock)** - 死锁检测、预防、避免
- **信号量(Semaphore)** - 二进制信号量、计数信号量
- **互斥锁(Mutex)** - 自旋锁、可重入锁
- **条件变量(Condition Variable)** - 等待/通知机制

## 核心概念

### 进程状态
- **新建(New)** - 进程正在被创建
- **就绪(Ready)** - 进程等待CPU分配
- **运行(Running)** - 进程正在执行
- **等待/阻塞(Waiting/Blocked)** - 进程等待某事件发生
- **终止(Terminated)** - 进程执行完成

### 内存管理策略
- **固定分区分配** - 内存划分为固定大小的分区
- **动态分区分配** - 根据进程需求动态分配内存
- **分页系统** - 将内存划分为固定大小的页
- **分段系统** - 根据程序的逻辑结构划分内存

### 文件系统类型
- **FAT(File Allocation Table)** - 文件分配表系统
- **ext2/ext3/ext4** - Linux扩展文件系统
- **NTFS** - Windows新技术文件系统
- **HFS+** - 苹果分层文件系统

### 调度算法对比
| 算法 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| FCFS | 简单公平 | 平均等待时间长 | 批处理系统 |
| SJF | 平均等待时间最短 | 可能导致饥饿 | 预知作业长度时 |
| RR | 响应时间好 | 上下文切换开销大 | 交互式系统 |
| 优先级 | 重要任务优先 | 低优先级任务饥饿 | 实时系统 |

## 性能指标

### CPU调度性能
- **CPU利用率** - CPU忙碌时间与总时间的比率
- **吞吐量** - 单位时间内完成的进程数
- **周转时间** - 从提交到完成的总时间
- **等待时间** - 在就绪队列中的等待时间
- **响应时间** - 从提交到首次响应的时间

### 内存管理性能
- **内存利用率** - 实际使用的内存与总内存的比率
- **缺页率** - 页面置换发生的频率
- **访问时间** - 内存访问的平均时间
- **碎片率** - 碎片内存与总内存的比率

## 学习建议

1. **理论与实践结合** - 理解概念后立即动手实现
2. **对比分析** - 比较不同算法的优缺点和适用场景
3. **性能评估** - 分析不同实现的时间和空间复杂度
4. **实际应用** - 思考在真实系统中如何应用这些概念
