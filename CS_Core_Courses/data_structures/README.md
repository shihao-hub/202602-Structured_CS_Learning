# 数据结构与算法 (Data Structures & Algorithms)

本目录包含了数据结构核心概念的Go语言实现，帮助你深入理解计算机科学中的基础数据结构。

## 目录结构

### [基础数据结构](./basic/)
- **数组(Array)** - 连续内存空间的线性数据结构
- **链表(Linked List)** - 动态内存分配的线性数据结构
- **栈(Stack)** - LIFO (后进先出) 的数据结构
- **队列(Queue)** - FIFO (先进先出) 的数据结构
- **哈希表(Hash Table)** - 基于键值对的快速查找结构

### [线性数据结构](./linear/)
- **动态数组(Dynamic Array)** - 自动扩容的数组实现
- **双端队列(Deque)** - 两端都可以插入和删除的队列
- **循环队列(Circular Queue)** - 环形缓冲区实现

### [树形结构](./tree/)
- **二叉搜索树(Binary Search Tree)** - 有序的二叉树
- **平衡二叉树(AVL Tree)** - 自平衡的二叉搜索树
- **红黑树(Red-Black Tree)** - 广泛使用的平衡二叉树
- **B树/B+树(B-Tree/B+ Tree)** - 多路搜索树，用于数据库索引
- **堆(Heap)** - 完全二叉树，用于优先队列
- **字典树(Trie)** - 字符串前缀匹配树

### [图结构](./graph/)
- **图的表示方法** - 邻接矩阵、邻接表
- **图的遍历** - DFS、BFS
- **最短路径算法** - Dijkstra、Floyd-Warshall
- **最小生成树** - Prim、Kruskal
- **拓扑排序** - 有向无环图的排序

### [算法](./algorithm/)
- **排序算法** - 冒泡、选择、插入、快速、归并、堆排序
- **搜索算法** - 线性搜索、二分搜索
- **递归与分治** - 经典分治算法
- **动态规划** - 最优化问题的解决方案
- **贪心算法** - 局部最优选择
- **回溯算法** - 深度优先搜索的应用

## 学习建议

1. **循序渐进** - 从基础数据结构开始，逐步学习复杂结构
2. **动手实践** - 运行每个示例，理解时间和空间复杂度
3. **对比分析** - 比较不同数据结构的优缺点和适用场景
4. **实际应用** - 思考在什么场景下使用哪种数据结构

## 复杂度分析

### 时间复杂度 (Time Complexity)
- **O(1)** - 常数时间
- **O(log n)** - 对数时间
- **O(n)** - 线性时间
- **O(n log n)** - 线性对数时间
- **O(n²)** - 平方时间

### 空间复杂度 (Space Complexity)
- **O(1)** - 常数空间
- **O(n)** - 线性空间
- **O(n²)** - 平方空间
