## 网络体系结构

### OSI七层参考模型
```
7. 应用层 (Application Layer)     - HTTP, FTP, SMTP, DNS
6. 表示层 (Presentation Layer)    - SSL/TLS, 数据加密
5. 会话层 (Session Layer)         - 会话管理、同步
4. 传输层 (Transport Layer)       - TCP, UDP
3. 网络层 (Network Layer)         - IP, ICMP, ARP
2. 数据链路层 (Data Link Layer)   - Ethernet, MAC
1. 物理层 (Physical Layer)        - 信号、传输介质
```

### TCP/IP四层模型
```
4. 应用层 (Application)           - HTTP, FTP, DNS
3. 传输层 (Transport)             - TCP, UDP
2. 网络层 (Internet)              - IP, ICMP, ARP
1. 网络接口层 (Network Interface) - Ethernet, WiFi
```

## 核心协议详解

### TCP协议特性
- **面向连接** - 三次握手建立连接
- **可靠传输** - 确认、重传、排序
- **流量控制** - 滑动窗口机制
- **拥塞控制** - 慢启动、拥塞避免
- **全双工通信** - 双向同时传输

### UDP协议特性
- **无连接** - 无需建立连接
- **不可靠传输** - 可能丢失、重复、乱序
- **开销小** - 头部仅8字节
- **实时性好** - 适合实时应用

### IP协议功能
- **路由选择** - 根据IP地址选择路由
- **分片重组** - 处理MTU限制
- **地址分配** - IPv4/IPv6地址管理
- **尽力交付** - 不保证可靠性

## 网络地址与端口

### IP地址分类
| 类型 | 范围 | 用途 |
|------|------|------|
| A类 | 1.0.0.0 - 126.255.255.255 | 大型网络 |
| B类 | 128.0.0.0 - 191.255.255.255 | 中型网络 |
| C类 | 192.0.0.0 - 223.255.255.255 | 小型网络 |
| D类 | 224.0.0.0 - 239.255.255.255 | 组播地址 |
| E类 | 240.0.0.0 - 255.255.255.255 | 保留地址 |

### 特殊地址
- **127.0.0.1** - 本地回环地址
- **0.0.0.0** - 任意地址
- **255.255.255.255** - 有限广播地址
- **169.254.x.x** - 链路本地地址

### 端口分配
- **熟知端口 (0-1023)** - 系统服务端口
- **注册端口 (1024-49151)** - 应用程序端口
- **动态端口 (49152-65535)** - 客户端临时端口

## 性能指标

### 网络性能指标
- **带宽 (Bandwidth)** - 最大传输速率
- **吞吐量 (Throughput)** - 实际传输速率
- **延迟 (Latency)** - 信号传输时间
- **抖动 (Jitter)** - 延迟变化
- **丢包率 (Packet Loss)** - 丢失数据包比例

### TCP性能指标
- **连接建立时间** - 三次握手时间
- **传输速率** - 数据传输速度
- **重传率** - 数据包重传比例
- **窗口大小** - 接收窗口大小

## 网络安全

### 基本安全措施
- **防火墙** - 过滤网络流量
- **加密传输** - SSL/TLS加密
- **身份认证** - 用户身份验证
- **访问控制** - 权限管理

### 常见攻击防护
- **DDoS攻击** - 分布式拒绝服务
- **中间人攻击** - 窃听和篡改
- **SQL注入** - 数据库攻击
- **XSS攻击** - 跨站脚本攻击
