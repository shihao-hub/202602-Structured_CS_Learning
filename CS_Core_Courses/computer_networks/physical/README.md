# 物理层 (Physical Layer)

## 408 考纲映射

本模块为理论概念性内容,涵盖计算机网络 408 考试中物理层的核心知识点。物理层不涉及代码实现,重点掌握以下理论知识:

## 1. 通信基础理论

### 1.1 Nyquist 定理 (奈奎斯特定理)
**理想低通信道的最高码元传输速率:**

```
理想信道: Rmax = 2 × W × log₂V (bps)
```

- **W**: 信道带宽 (Hz)
- **V**: 信号电平数
- **意义**: 无噪声信道的极限传输速率

**示例:**
- 信道带宽 3000 Hz, 4 电平信号
- Rmax = 2 × 3000 × log₂4 = 12000 bps

### 1.2 Shannon 定理 (香农定理)
**有噪声信道的极限数据传输速率:**

```
Shannon 容量: C = W × log₂(1 + S/N) (bps)
```

- **W**: 信道带宽 (Hz)
- **S/N**: 信噪比 (Signal-to-Noise Ratio)
- **常用单位**: 信噪比分贝值 = 10 × log₁₀(S/N) (dB)

**示例:**
- 信道带宽 3000 Hz, 信噪比 30 dB
- S/N = 10^(30/10) = 1000
- C = 3000 × log₂(1001) ≈ 30000 bps

**408 考点:**
- Nyquist 给出码元速率上限 (无噪声)
- Shannon 给出比特速率上限 (有噪声)
- 实际信道受两者共同限制

## 2. 编码方式

### 2.1 数字数据编码为数字信号

#### NRZ (Non-Return to Zero, 不归零编码)
- **高电平表示 1, 低电平表示 0**
- 优点: 简单
- 缺点: 连续 0/1 时无时钟同步信息

```
数据:  1  0  1  1  0  0  1
NRZ:  ‾‾|__|‾‾‾‾|__|‾‾
```

#### Manchester 编码 (曼彻斯特编码)
- **每个码元中间都有跳变: 低→高表示 1, 高→低表示 0**
- 优点: 自同步 (每位都有跳变)
- 缺点: 带宽需求是 NRZ 的 2 倍
- **应用**: 传统以太网 (10BASE-T)

```
数据:       1     0     1     1
Manchester: _|‾‾|_  ‾‾|_|‾  _|‾‾|_  _|‾‾|_
           (低→高) (高→低) (低→高) (低→高)
```

#### Differential Manchester 编码 (差分曼彻斯特编码)
- **每位中间都有跳变 (用于同步), 位开始处有无跳变表示数据**
- 1: 位开始处**无跳变**
- 0: 位开始处**有跳变**
- 优点: 抗干扰性强
- **应用**: 令牌环网络

```
数据:    1       0       1       0
Diff-M:  ‾‾|_|‾  |_|‾‾|_  ‾‾|_|‾  |_|‾‾|_
        (无跳变) (有跳变) (无跳变) (有跳变)
```

### 2.2 模拟数据编码
- **PCM (脉冲编码调制)**: 采样 → 量化 → 编码
- **采样定理**: 采样频率 ≥ 2 × 信号最高频率

## 3. 多路复用技术

### 3.1 FDM (Frequency Division Multiplexing, 频分复用)
- 不同用户占用不同频带
- 所有用户同时传输
- 应用: 广播、有线电视

### 3.2 TDM (Time Division Multiplexing, 时分复用)
- 不同用户占用不同时隙
- 用户轮流使用全部带宽
- **统计时分复用 (STDM)**: 动态分配时隙,提高利用率

### 3.3 WDM (Wavelength Division Multiplexing, 波分复用)
- 光纤通信中的频分复用
- 不同波长 (颜色) 的光信号同时传输
- **DWDM (密集波分复用)**: 更多波长,更高容量

### 3.4 CDM (Code Division Multiplexing, 码分复用)
- 每个用户使用唯一的码片序列
- 所有用户同时使用相同频带
- **CDMA (码分多址)**: 应用于 3G 移动通信
- **特点**: 
  - 码片序列正交
  - 抗干扰能力强
  - 频谱利用率高

**码片序列示例 (408 常考):**
```
站点 A 码片: (+1, -1, +1, +1, -1, -1, +1, -1)
站点 B 码片: (+1, +1, -1, +1, +1, +1, -1, -1)

A 发送 1: 发送码片序列本身
A 发送 0: 发送码片序列的反码
```

**内积运算 (解码):**
```
接收信号 · 码片序列 / 码片长度 = 数据
结果 = 1  → 发送的是 1
结果 = -1 → 发送的是 0
结果 = 0  → 未发送数据
```

## 4. 传输媒体

### 4.1 导向传输媒体

#### 双绞线 (Twisted Pair)
- **UTP (非屏蔽双绞线)**: 最常用,价格低
- **STP (屏蔽双绞线)**: 抗干扰性强
- **分类**: Cat5 (100 Mbps), Cat5e (1 Gbps), Cat6 (10 Gbps)
- **最大传输距离**: 100 米 (以太网)

#### 同轴电缆 (Coaxial Cable)
- 抗干扰性强于双绞线
- 应用: 有线电视网络

#### 光纤 (Optical Fiber)
- **单模光纤**: 传输距离远 (数十公里), 价格高
- **多模光纤**: 传输距离近 (数百米), 价格低
- **优点**: 
  - 带宽大 (Gbps - Tbps)
  - 传输距离远
  - 抗电磁干扰
  - 保密性好
- **缺点**: 成本高,不易弯曲

### 4.2 非导向传输媒体 (无线传输)

#### 无线电波
- **频段**: 长波、中波、短波、微波
- 传输距离远,穿透能力强
- 应用: 广播、通信

#### 微波
- **地面微波**: 直线传输,中继站
- **卫星微波**: 覆盖范围广,延迟大 (约 270 ms)

#### 红外线与激光
- 红外线: 短距离,不能穿墙
- 激光: 方向性强,易受天气影响

## 5. 物理层设备

### 5.1 中继器 (Repeater)
- **功能**: 信号再生与放大
- **工作层次**: 物理层
- **作用**: 延长传输距离

### 5.2 集线器 (Hub)
- **本质**: 多端口中继器
- **工作层次**: 物理层
- **特点**: 
  - 广播方式 (半双工)
  - 共享带宽
  - 冲突域不分隔
  - 已被交换机取代

## 6. 408 常考知识点

### 重点公式
1. **Nyquist 定理**: Rmax = 2W log₂V
2. **Shannon 定理**: C = W log₂(1 + S/N)
3. **信噪比换算**: SNR(dB) = 10 log₁₀(S/N)
4. **传播时延**: t = 距离 / 传播速度
5. **发送时延**: t = 数据长度 / 发送速率

### 重点概念
- ✓ Nyquist 与 Shannon 定理的区别与应用
- ✓ Manchester 编码的特点 (每位中间都有跳变)
- ✓ 码分复用 (CDMA) 的内积运算
- ✓ 光纤的优点 (带宽大、距离远、抗干扰)
- ✓ 中继器与集线器的物理层特性

### 典型题型
1. 计算题: 给定带宽和信噪比,求最大传输速率
2. 编码题: 绘制 NRZ、Manchester、Differential Manchester 波形
3. 复用题: CDMA 码片序列的内积运算
4. 对比题: 单模光纤 vs 多模光纤,FDM vs TDM

## 学习建议

1. **掌握公式**: Nyquist 和 Shannon 定理是必考点
2. **理解编码**: 能够绘制三种编码方式的波形图
3. **熟悉复用**: 重点掌握 CDMA 的码片序列计算
4. **记忆参数**: 
   - 双绞线最大距离 100m
   - 光速 3×10⁸ m/s (真空), 2×10⁸ m/s (介质)
   - 以太网信道利用率公式
5. **结合习题**: 多做 408 真题中的物理层计算题
