## 📌 常见问题

**Q1：二次型与实对称矩阵有什么关系？**  
A：任何二次型都可以表示为 f = x^T A x，其中 A 是实对称矩阵。反之，任何实对称矩阵都对应一个二次型。

**Q2：配方法和正交变换法哪个更好？**  
A：
- **配方法**：适合手算，步骤清晰，但不能求出变换矩阵
- **正交变换法**：需要求特征值和特征向量，能给出具体的正交变换矩阵

**Q3：如何判断矩阵是否可以对角化？**  
A：
- 实对称矩阵**一定可以对角化**（考研高频结论）
- 一般矩阵：有 n 个线性无关的特征向量 ⇔ 可对角化

**Q4：线性方程组的解与秩有什么关系？**  
A：
- 有唯一解 ⇔ rank(A) = rank(A|b) = n（未知数个数）
- 有无穷多解 ⇔ rank(A) = rank(A|b) < n
- 无解 ⇔ rank(A) < rank(A|b)

## 🔍 核心定理速查

### 实对称矩阵性质
1. 特征值都是实数
2. 不同特征值对应的特征向量正交
3. 必可相似对角化：A = QΛQ^T（Q 为正交矩阵）

### 正定矩阵判定
设 A 为 n 阶实对称矩阵，下列命题等价：
1. A 正定
2. A 的所有特征值 > 0
3. A 的各阶顺序主子式 > 0
4. A 合同于单位矩阵（存在可逆矩阵 C 使 A = C^T C）
5. 二次型 x^T A x 对任意 x ≠ 0 都 > 0

### 惯性定理
二次型的正惯性指数、负惯性指数、符号差在合同变换下保持不变。
