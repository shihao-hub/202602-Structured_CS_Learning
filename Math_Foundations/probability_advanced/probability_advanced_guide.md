## 1. 多维随机变量深化

### 1.1 二维随机变量的分布

#### 联合分布
设 (X, Y) 是二维随机变量，其联合分布函数为：
```
F(x, y) = P(X ≤ x, Y ≤ y)
```

**离散型**：联合概率质量函数
```
P(X = xᵢ, Y = yⱼ) = pᵢⱼ
```

**连续型**：联合概率密度函数
```
P((X,Y) ∈ D) = ∬_D f(x,y) dxdy
```

#### 边缘分布
从联合分布求单个变量的分布。

**离散型**：
```
P(X = xᵢ) = Σⱼ pᵢⱼ
P(Y = yⱼ) = Σᵢ pᵢⱼ
```

**连续型**：
```
f_X(x) = ∫_{-∞}^{∞} f(x,y) dy
f_Y(y) = ∫_{-∞}^{∞} f(x,y) dx
```

#### 条件分布
在一个变量取某值的条件下，另一个变量的分布。

**离散型**：
```
P(Y = yⱼ | X = xᵢ) = pᵢⱼ / P(X = xᵢ)
```

**连续型**：
```
f_{Y|X}(y|x) = f(x,y) / f_X(x)
```

#### 典型例题
**例1**：设 (X,Y) 的联合密度函数为：
```
f(x,y) = { 2, 0 ≤ x ≤ y ≤ 1
         { 0, 其他
```

求：(1) 边缘密度 f_X(x), f_Y(y)；(2) 条件密度 f_{Y|X}(y|x)

**解**：

**(1) 边缘密度**
```
f_X(x) = ∫_{-∞}^{∞} f(x,y) dy = ∫_x^1 2 dy = 2(1-x)  (0 ≤ x ≤ 1)

f_Y(y) = ∫_{-∞}^{∞} f(x,y) dx = ∫_0^y 2 dx = 2y  (0 ≤ y ≤ 1)
```

**(2) 条件密度**
```
f_{Y|X}(y|x) = f(x,y) / f_X(x) = 2 / (2(1-x)) = 1/(1-x)  (x ≤ y ≤ 1)
```

这是在 [x, 1] 上的均匀分布。

### 1.2 随机变量的独立性

#### 定义
X 与 Y 独立 ⇔ F(x, y) = F_X(x) · F_Y(y) 对所有 (x, y) 成立

**等价条件**：
- 离散型：p_{ij} = P(X=x_i) · P(Y=y_j) 对所有 i, j
- 连续型：f(x, y) = f_X(x) · f_Y(y) 对几乎所有 (x, y)

#### 判别技巧
若联合密度可以分解为 **f(x,y) = g(x) · h(y)**，且定义域为矩形，则 X 与 Y 独立。

**注意**：若定义域不是矩形（如三角形），即使能分解也不独立！

#### 典型例题
**例2**：判断下列情况下 X 与 Y 是否独立：

(a) f(x,y) = 4xy, 0 ≤ x ≤ 1, 0 ≤ y ≤ 1

(b) f(x,y) = 2, 0 ≤ x ≤ y ≤ 1

**解**：

(a) 定义域是矩形 [0,1]×[0,1]，且 f(x,y) = (2x)(2y)，**独立**。

(b) 定义域是三角形（非矩形），**不独立**。

### 1.3 协方差与相关系数

#### 协方差
```
Cov(X, Y) = E[(X - E(X))(Y - E(Y))] = E(XY) - E(X)E(Y)
```

**性质**：
1. Cov(X, X) = D(X)（方差）
2. Cov(X, Y) = Cov(Y, X)（对称性）
3. Cov(aX + b, cY + d) = ac Cov(X, Y)
4. D(X ± Y) = D(X) + D(Y) ± 2Cov(X, Y)

#### 相关系数
```
ρ_{XY} = Cov(X, Y) / (σ_X σ_Y)
```

**性质**：
1. |ρ| ≤ 1
2. |ρ| = 1 ⇔ 存在常数 a, b 使 P(Y = aX + b) = 1（线性相关）
3. ρ = 0 ⇔ X 与 Y 不相关
4. 独立 ⇒ 不相关（但反之不一定）

#### 不相关 vs 独立

| 关系 | 定义 | 意义 |
|------|------|------|
| 独立 | P(X∈A, Y∈B) = P(X∈A)P(Y∈B) | 完全无关 |
| 不相关 | Cov(X,Y) = 0 或 ρ = 0 | 无线性关系 |

**重要结论**：
- 独立 ⇒ 不相关
- 不相关 ⇏ 独立（除非 (X,Y) 服从二维正态）
- 对于二维正态分布：独立 ⇔ 不相关

#### 典型例题
**例3**：设 X ~ U(0, 1)，Y = X²，求 ρ_{XY}。

**解**：
```
E(X) = 1/2, E(X²) = 1/3, E(X³) = 1/4

E(Y) = E(X²) = 1/3

E(XY) = E(X³) = 1/4

Cov(X, Y) = E(XY) - E(X)E(Y) = 1/4 - (1/2)(1/3) = 1/12

D(X) = E(X²) - [E(X)]² = 1/3 - 1/4 = 1/12

D(Y) = E(X⁴) - [E(X²)]² = 1/5 - 1/9 = 4/45

ρ_{XY} = Cov(X,Y) / √(D(X)D(Y)) = (1/12) / √((1/12)(4/45))
       = (1/12) / (2/(6√5)) = √5/2 ≈ 0.745
```

虽然 ρ ≠ 0（相关），但 X 与 Y = X² 显然不独立。

### 1.4 多元正态分布

#### 二维正态分布
(X, Y) ~ N(μ₁, μ₂, σ₁², σ₂², ρ)，密度函数：
```
f(x,y) = 1/(2πσ₁σ₂√(1-ρ²)) exp{-Q/(2(1-ρ²))}
```

其中：
```
Q = [(x-μ₁)/σ₁]² - 2ρ[(x-μ₁)/σ₁][(y-μ₂)/σ₂] + [(y-μ₂)/σ₂]²
```

#### 性质
1. 边缘分布：X ~ N(μ₁, σ₁²), Y ~ N(μ₂, σ₂²)
2. 条件分布：Y|X=x ~ N(μ₂ + ρ(σ₂/σ₁)(x-μ₁), σ₂²(1-ρ²))
3. **独立 ⇔ 不相关 ⇔ ρ = 0**（只对正态分布成立）
4. 线性组合仍为正态：aX + bY ~ N(...)

---

### 1.5 考试重点

**高频考点**：
- 由联合分布求边缘分布、条件分布（★★★）
- 判断独立性（★★★）
- 计算协方差与相关系数（★★★）
- 二维正态分布的性质应用（★★）

**易错点**：
1. 定义域非矩形时判断独立性
2. 不相关与独立混淆
3. 条件分布公式记错
4. 协方差计算时忘记 E(XY) - E(X)E(Y)

## 2. 三大抽样分布

### 2.1 χ²分布（卡方分布）

#### 定义
设 X₁, X₂, ..., Xₙ 独立同分布于 N(0,1)，则：
```
χ² = X₁² + X₂² + ... + Xₙ²
```

服从**自由度为 n 的 χ²分布**，记作 χ² ~ χ²(n)。

#### 性质
1. **可加性**：若 X ~ χ²(m), Y ~ χ²(n) 独立，则 X + Y ~ χ²(m+n)
2. E(χ²) = n, D(χ²) = 2n
3. 当 n → ∞ 时，(χ²-n)/√(2n) → N(0,1)

#### 分位点
χ²_α(n)：满足 P(χ² > χ²_α(n)) = α

**查表使用**：给定 n 和 α，查 χ²分布表得 χ²_α(n)。

---

### 2.2 t 分布

#### 定义
设 X ~ N(0,1), Y ~ χ²(n) 且独立，则：
```
T = X / √(Y/n)
```

服从**自由度为 n 的 t 分布**，记作 T ~ t(n)。

#### 性质
1. t 分布是**对称分布**（关于0对称）
2. 当 n → ∞ 时，t(n) → N(0,1)
3. t 分布的尾部比正态分布更"厚"
4. E(T) = 0（n > 1），D(T) = n/(n-2)（n > 2）

#### 分位点
t_α(n)：满足 P(T > t_α(n)) = α

**对称性**：t_{1-α}(n) = -t_α(n)

---

### 2.3 F 分布

#### 定义
设 X ~ χ²(m), Y ~ χ²(n) 且独立，则：
```
F = (X/m) / (Y/n)
```

服从**第一自由度为 m、第二自由度为 n 的 F 分布**，记作 F ~ F(m, n)。

#### 性质
1. 若 F ~ F(m, n)，则 1/F ~ F(n, m)（倒数关系）
2. 若 T ~ t(n)，则 T² ~ F(1, n)
3. F 分布是**非对称分布**

#### 分位点
F_α(m, n)：满足 P(F > F_α(m, n)) = α

**倒数关系**：F_{1-α}(m, n) = 1 / F_α(n, m)

---

### 2.4 正态总体的抽样分布

设 X₁, X₂, ..., Xₙ 是来自 N(μ, σ²) 的简单随机样本。

#### 单个正态总体

**定理1**：样本均值与样本方差独立
```
X̄ ~ N(μ, σ²/n)  （样本均值的分布）

(n-1)S²/σ² ~ χ²(n-1)  （样本方差的分布）
```

且 X̄ 与 S² 相互独立。

**定理2**：
```
(X̄ - μ) / (σ/√n) ~ N(0, 1)  （σ已知）

(X̄ - μ) / (S/√n) ~ t(n-1)  （σ未知，用S估计）
```

#### 两个正态总体

设 X ~ N(μ₁, σ₁²)（样本量 n₁），Y ~ N(μ₂, σ₂²)（样本量 n₂），且独立。

**定理3**：
```
(X̄ - Ȳ) ~ N(μ₁ - μ₂, σ₁²/n₁ + σ₂²/n₂)
```

**定理4**：
```
F = S₁²/σ₁² / (S₂²/σ₂²) ~ F(n₁-1, n₂-1)
```

特别地，若 σ₁² = σ₂²，则 F = S₁²/S₂² ~ F(n₁-1, n₂-1)。

---

### 2.5 考试重点

**高频考点**：
- 三大分布的定义与构造（★★★）
- 正态总体下的抽样分布定理（★★★）
- 分位点的查表与使用（★★）

**易错点**：
1. t 分布自由度是 n-1 而不是 n
2. F 分布两个自由度的顺序不能颠倒
3. 混淆σ已知（用 N(0,1)）和σ未知（用 t）的情况

## 3. 参数估计

### 3.1 点估计

#### 矩估计法

**思想**：用样本矩估计总体矩。

**步骤**：
1. 写出总体的 k 阶矩关于参数的表达式
2. 用样本 k 阶矩代替总体 k 阶矩
3. 解方程组得到参数估计

**公式**：
```
μ̂ = X̄ = (1/n) Σ Xᵢ  （估计期望）

σ̂² = (1/n) Σ (Xᵢ - X̄)²  （估计方差）
```

#### 最大似然估计（MLE）

**思想**：选择使得"观测到当前样本"概率最大的参数值。

**步骤**：
1. 写出似然函数：L(θ) = ∏ f(xᵢ; θ)
2. 取对数：ln L(θ) = Σ ln f(xᵢ; θ)
3. 求导：d(ln L)/dθ = 0
4. 解方程得 θ̂_MLE

**典型例题**
**例4**：设 X₁, ..., Xₙ ~ U(0, θ)，求 θ 的 MLE。

**解**：
密度函数：f(x; θ) = 1/θ, 0 ≤ x ≤ θ

似然函数：
```
L(θ) = (1/θ)ⁿ,  θ ≥ max{X₁, ..., Xₙ}
```

L(θ) 关于 θ 单调递减，故在 θ = max{Xᵢ} 时最大。

```
θ̂_MLE = max{X₁, X₂, ..., Xₙ}
```

---

### 3.2 估计量的评价标准

#### 无偏性
```
E(θ̂) = θ
```

**例**：X̄ 是 μ 的无偏估计，S² 是 σ² 的无偏估计。

#### 有效性
设 θ̂₁ 和 θ̂₂ 都是 θ 的无偏估计，若 D(θ̂₁) < D(θ̂₂)，则 θ̂₁ 比 θ̂₂ **更有效**。

#### 一致性（相合性）
```
θ̂ₙ →^P θ  （当 n → ∞）
```

即：对任意 ε > 0，lim P(|θ̂ₙ - θ| < ε) = 1。

---

### 3.3 区间估计

#### 置信区间
设 θ 是待估参数，若存在统计量 θ̂₁ 和 θ̂₂，使得：
```
P(θ̂₁ < θ < θ̂₂) = 1 - α
```

则 (θ̂₁, θ̂₂) 称为 θ 的**置信水平为 1-α 的置信区间**。

#### 正态总体均值的置信区间

**情况1：σ²已知**
```
置信区间：X̄ ± z_{α/2} · σ/√n
```

**情况2：σ²未知**
```
置信区间：X̄ ± t_{α/2}(n-1) · S/√n
```

#### 正态总体方差的置信区间
```
置信区间：((n-1)S² / χ²_{α/2}(n-1), (n-1)S² / χ²_{1-α/2}(n-1))
```

---

### 3.4 考试重点

**高频考点**：
- 矩估计与 MLE 的计算（★★★）
- 估计量无偏性的验证（★★★）
- 置信区间的构造（★★）

**易错点**：
1. MLE 求导时忘记对数似然
2. 均匀分布等特殊情况下 MLE 需要用单调性
3. 置信区间端点公式记错（特别是方差的）

## 4. 假设检验深化

### 4.1 假设检验的基本思想

#### 原假设与备择假设
- **原假设 H₀**：待检验的假设
- **备择假设 H₁**：与 H₀ 对立的假设

#### 两类错误

| 错误类型 | 定义 | 概率 |
|----------|------|------|
| 第一类错误 | H₀为真时拒绝H₀（弃真） | α（显著性水平） |
| 第二类错误 | H₀为假时接受H₀（取伪） | β |

#### 检验步骤
1. 建立假设：H₀ vs H₁
2. 选择检验统计量
3. 给定显著性水平 α
4. 确定拒绝域
5. 计算统计量的值
6. 做出判断

---

### 4.2 单个正态总体的假设检验

#### 均值的检验（μ）

**情况1：σ²已知，H₀: μ = μ₀**
```
检验统计量：Z = (X̄ - μ₀) / (σ/√n) ~ N(0,1)

拒绝域（双侧）：|Z| > z_{α/2}
拒绝域（右侧）：Z > z_α
拒绝域（左侧）：Z < -z_α
```

**情况2：σ²未知，H₀: μ = μ₀**
```
检验统计量：T = (X̄ - μ₀) / (S/√n) ~ t(n-1)

拒绝域（双侧）：|T| > t_{α/2}(n-1)
```

#### 方差的检验（σ²）

**H₀: σ² = σ₀²**
```
检验统计量：χ² = (n-1)S² / σ₀² ~ χ²(n-1)

拒绝域（双侧）：χ² < χ²_{1-α/2}(n-1) 或 χ² > χ²_{α/2}(n-1)
```

---

### 4.3 两个正态总体的假设检验

#### 均值差的检验

**H₀: μ₁ = μ₂（或 μ₁ - μ₂ = 0）**

**情况1：σ₁², σ₂²已知**
```
Z = (X̄ - Ȳ) / √(σ₁²/n₁ + σ₂²/n₂) ~ N(0,1)
```

**情况2：σ₁² = σ₂² = σ²未知**
```
T = (X̄ - Ȳ) / (Sω √(1/n₁ + 1/n₂)) ~ t(n₁ + n₂ - 2)

其中：Sω² = ((n₁-1)S₁² + (n₂-1)S₂²) / (n₁ + n₂ - 2)  （合并方差）
```

#### 方差比的检验

**H₀: σ₁² = σ₂²**
```
F = S₁² / S₂² ~ F(n₁-1, n₂-1)

拒绝域（双侧）：F < F_{1-α/2}(n₁-1, n₂-1) 或 F > F_{α/2}(n₁-1, n₂-1)
```

---

### 4.4 p 值

#### 定义
**p 值**是在原假设为真的条件下，出现比观测样本更极端情况的概率。

#### 判断准则
- p < α：拒绝 H₀
- p ≥ α：不能拒绝 H₀

#### p 值的优势
- 提供连续的证据强度
- 不需要预先设定 α
- 更符合科学研究习惯

---

### 4.5 检验的功效函数

#### 功效函数
```
β(θ) = P(拒绝H₀ | θ)
```

- 当 θ ∈ H₀ 时，β(θ) = α（第一类错误概率）
- 当 θ ∈ H₁ 时，β(θ) = 1 - P(第二类错误) = 检验的功效

#### 功效曲线
以 θ 为横轴，β(θ) 为纵轴绘制的曲线，反映检验在不同参数下的表现。

---

### 4.6 考试重点

**高频考点**：
- 单个正态总体均值/方差的检验（★★★）
- 两个正态总体均值差/方差比的检验（★★★）
- 两类错误的理解（★★）
- p 值的计算与应用（★★）

**易错点**：
1. 混淆单侧检验与双侧检验的拒绝域
2. 方差检验用 χ² 还是 F 搞混
3. t 检验的自由度（n-1 或 n₁+n₂-2）

## 5. 一元线性回归

### 5.1 回归模型

#### 模型假设
```
Y = β₀ + β₁X + ε
```

其中：
- β₀：截距
- β₁：斜率（回归系数）
- ε ~ N(0, σ²)：随机误差

---

### 5.2 最小二乘估计

#### 估计公式
```
β̂₁ = Σ(Xᵢ - X̄)(Yᵢ - Ȳ) / Σ(Xᵢ - X̄)² = L_{xy} / L_{xx}

β̂₀ = Ȳ - β̂₁X̄
```

其中：
```
L_{xx} = Σ(Xᵢ - X̄)² = Σ Xᵢ² - n X̄²
L_{xy} = Σ(Xᵢ - X̄)(Yᵢ - Ȳ) = Σ XᵢYᵢ - n X̄Ȳ
```

---

### 5.3 回归方程的检验

#### 方差分析表

| 来源 | 平方和 | 自由度 | 均方 | F 值 |
|------|--------|--------|------|------|
| 回归 | SSR = Σ(Ŷᵢ - Ȳ)² | 1 | MSR = SSR/1 | F = MSR/MSE |
| 残差 | SSE = Σ(Yᵢ - Ŷᵢ)² | n-2 | MSE = SSE/(n-2) | |
| 总计 | SST = Σ(Yᵢ - Ȳ)² | n-1 | | |

**F 检验**：H₀: β₁ = 0
```
F = MSR / MSE ~ F(1, n-2)

拒绝域：F > F_α(1, n-2)
```

#### 决定系数 R²
```
R² = SSR / SST = 1 - SSE / SST
```

R² 越接近 1，拟合越好。

---

### 5.4 残差分析

#### 残差
```
eᵢ = Yᵢ - Ŷᵢ
```

#### 残差图
- **残差 vs 预测值**：检验等方差性
- **残差 vs 自变量**：检验线性关系
- **残差的正态QQ图**：检验正态性

---
