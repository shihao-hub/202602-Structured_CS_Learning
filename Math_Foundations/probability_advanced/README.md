## 📌 常见问题

**Q1：三大抽样分布有什么区别？**  
A：
- **χ²分布**：正态变量平方和的分布
- **t 分布**：样本均值的分布（方差未知时）
- **F 分布**：两个χ²分布之比的分布

**Q2：MLE 和矩估计哪个更好？**  
A：MLE 在大样本下具有最优性（渐近有效），但计算复杂；矩估计简单直观，但效率较低。

**Q3：p 值和显著性水平α有什么关系？**  
A：p 值是在原假设下观测到当前样本或更极端情况的概率。若 p < α，拒绝原假设。

**Q4：线性回归的假设有哪些？**  
A：
1. 线性关系
2. 误差独立同分布
3. 误差服从正态分布
4. 等方差性

## 🔍 核心公式速查

### 三大抽样分布
```
χ²(n) = X₁² + X₂² + ... + Xₙ²  (Xᵢ ~ N(0,1) 独立)

t(n) = Z / √(χ²(n)/n)  (Z ~ N(0,1), χ²(n) 独立)

F(n₁, n₂) = (χ²(n₁)/n₁) / (χ²(n₂)/n₂)  (两个χ²独立)
```

### 最大似然估计
```
L(θ) = ∏ f(xᵢ; θ)  (似然函数)

θ̂_MLE = arg max L(θ)  或  arg max ln L(θ)
```

### 置信区间
```
P(θ̂ - t_(α/2) · SE < θ < θ̂ + t_(α/2) · SE) = 1 - α
```
