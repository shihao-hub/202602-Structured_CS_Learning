## 1. 矩阵与向量

### 1.1 矩阵的基本概念
- 矩阵是按矩形排列的数表
- m×n矩阵有m行n列
- 方阵：行数和列数相等

### 1.2 矩阵运算
- 加法：对应元素相加
- 数乘：每个元素乘以标量
- 乘法：行乘列的规则
- 转置：行列互换

### 1.3 向量
- 向量是特殊的矩阵（一维）
- 向量的模（长度）
- 向量的内积（点积）
- 向量的外积（叉积）

## 2. 线性方程组

### 2.1 线性方程组的形式
- 标准形式：Ax = b
- 增广矩阵：[A|b]
- 齐次方程组：Ax = 0

### 2.2 解法
- 高斯消元法
- 矩阵的行简化阶梯形
- 逆矩阵法：x = A⁻¹b
- 克拉默法则

### 2.3 解的存在性
- 唯一解：满秩方阵
- 无穷解：秩 < 未知数个数
- 无解：矛盾方程

## 3. 向量空间

### 3.1 向量空间的定义
- 非空集合V，满足8条公理
- 子空间：对加法和数乘封闭的子集

### 3.2 基与维度
- 线性无关
- 生成空间
- 基：线性无关的生成集
- 维度：基中向量的个数

### 3.3 内积空间
- 内积的定义和性质
- 正交性
- 格拉姆-施密特正交化

## 4. 特征值与特征向量

### 4.1 定义
- 特征值λ和特征向量v：Av = λv
- 特征方程：det(A - λI) = 0
- 特征多项式

### 4.2 性质
- 特征值的和等于迹
- 特征值的积等于行列式
- 不同特征值对应的特征向量线性无关

### 4.3 对角化
- 可对角化矩阵的条件
- 相似对角形
- 应用：矩阵幂的计算

## 5. 矩阵分解

### 5.1 LU分解
- 下三角矩阵L和上三角矩阵U
- A = LU
- 用于解线性方程组

### 5.2 QR分解
- 正交矩阵Q和上三角矩阵R
- A = QR
- 用于最小二乘问题

### 5.3 奇异值分解(SVD)
- A = UΣVᵀ
- 任意矩阵都可以分解
- 应用：降维、压缩

## 6. 线性变换

### 6.1 线性变换的定义
- T(u + v) = T(u) + T(v)
- T(cu) = cT(u)
- 矩阵表示

### 6.2 线性变换的性质
- 核空间（零空间）
- 像空间（值域）
- 秩-零化度定理

### 6.3 常见线性变换
- 旋转变换
- 缩放变换
- 投影变换
- 反射变换
