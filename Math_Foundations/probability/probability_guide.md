## 1. 概率基础

### 1.1 概率的定义
- 古典定义：P(A) = 有利情况数 / 所有可能情况数
- 几何定义：P(A) = 有利区域长度 / 总区域长度
- 频率定义：P(A) = 事件A发生的频率极限

### 1.2 概率的基本性质
- 非负性：P(A) ≥ 0
- 规范性：P(Ω) = 1
- 可列可加性：若A₁, A₂, ...互斥，则P(∪Aᵢ) = ΣP(Aᵢ)

### 1.3 概率的计算
- 加法公式：P(A∪B) = P(A) + P(B) - P(A∩B)
- 互补事件：P(A') = 1 - P(A)
- 容斥原理：多事件的并集概率

## 2. 条件概率与独立性

### 2.1 条件概率
- 定义：P(A|B) = P(A∩B) / P(B) (P(B) > 0)
- 乘法公式：P(A∩B) = P(A|B)P(B) = P(B|A)P(A)

### 2.2 贝叶斯定理
- 公式：P(A|B) = P(B|A)P(A) / P(B)
- 全概率公式：P(B) = ΣP(B|Aᵢ)P(Aᵢ)

### 2.3 独立性
- 定义：P(A∩B) = P(A)P(B)
- 等价条件：P(A|B) = P(A), P(B|A) = P(B)

## 3. 随机变量

### 3.1 随机变量的定义
- 离散型：取值可数的随机变量
- 连续型：取值连续的随机变量

### 3.2 分布函数
- 累积分布函数：F(x) = P(X ≤ x)
- 概率质量函数（离散型）：p(x) = P(X = x)
- 概率密度函数（连续型）：f(x) = F'(x)
