## 1. 曲线积分

### 1.1 第一型曲线积分（对弧长的曲线积分）

#### 概念
设 L 为平面或空间中的光滑曲线，f(x,y) 或 f(x,y,z) 在 L 上有界，则：

**平面曲线**：
```
∫_L f(x,y) ds
```

**空间曲线**：
```
∫_L f(x,y,z) ds
```

#### 物理意义
- 表示曲线形构件的质量（f 为线密度）
- 计算曲线的质心、转动惯量

#### 计算方法

**方法一：参数方程法**

若曲线 L 的参数方程为：
```
x = φ(t)
y = ψ(t)    (α ≤ t ≤ β)
z = ω(t)    [空间曲线]
```

则：
```
平面：∫_L f(x,y) ds = ∫_α^β f(φ(t), ψ(t)) √[φ'(t)² + ψ'(t)²] dt

空间：∫_L f(x,y,z) ds = ∫_α^β f(φ(t), ψ(t), ω(t)) √[φ'(t)² + ψ'(t)² + ω'(t)²] dt
```

**方法二：直角坐标方程**

若曲线 L: y = y(x), a ≤ x ≤ b，则：
```
∫_L f(x,y) ds = ∫_a^b f(x, y(x)) √[1 + y'(x)²] dx
```

#### 性质
1. **线性性**：∫_L [αf + βg] ds = α∫_L f ds + β∫_L g ds
2. **路径可加性**：∫_{L1+L2} f ds = ∫_{L1} f ds + ∫_{L2} f ds
3. **与路径方向无关**

#### 典型例题
**例1**：计算 ∫_L (x² + y²) ds，其中 L 是圆周 x² + y² = a²。

**解**：
参数方程：x = a cos t, y = a sin t, 0 ≤ t ≤ 2π

```
ds = √[(dx/dt)² + (dy/dt)²] dt = √[a²sin²t + a²cos²t] dt = a dt

∫_L (x² + y²) ds = ∫_0^{2π} a² · a dt = a³ ∫_0^{2π} dt = 2πa³
```

### 1.2 第二型曲线积分（对坐标的曲线积分）

#### 概念
设向量场 F = (P, Q) 或 F = (P, Q, R)，有向曲线 L，则：

**平面曲线**：
```
∫_L P dx + Q dy
```

**空间曲线**：
```
∫_L P dx + Q dy + R dz
```

#### 物理意义
- 变力沿曲线做功：W = ∫_L F·dr
- 流体的环流量

#### 计算方法

**参数方程法**

若 L: x = φ(t), y = ψ(t), (α ≤ t ≤ β)，则：
```
∫_L P dx + Q dy = ∫_α^β [P(φ(t), ψ(t))φ'(t) + Q(φ(t), ψ(t))ψ'(t)] dt
```

**注意**：积分值与路径方向有关！

若改变路径方向，积分值变号：
```
∫_{-L} P dx + Q dy = -∫_L P dx + Q dy
```

#### 两类曲线积分的关系
```
∫_L P dx + Q dy + R dz = ∫_L (P cos α + Q cos β + R cos γ) ds
```
其中 (cos α, cos β, cos γ) 是 L 的切向量方向余弦。

#### 典型例题
**例2**：计算 ∫_L y dx - x dy，其中 L 是从点 (1,0) 沿单位圆逆时针到点 (0,1) 的圆弧。

**解**：
参数方程：x = cos t, y = sin t, 0 ≤ t ≤ π/2

```
dx = -sin t dt, dy = cos t dt

∫_L y dx - x dy = ∫_0^{π/2} [sin t (-sin t) - cos t (cos t)] dt
                 = ∫_0^{π/2} [-(sin²t + cos²t)] dt
                 = ∫_0^{π/2} (-1) dt = -π/2
```

## 2. 格林公式

### 2.1 格林公式的表述

设闭区域 D 由分段光滑的曲线 L 围成，函数 P(x,y) 和 Q(x,y) 在 D 上具有一阶连续偏导数，则：

```
∮_L P dx + Q dy = ∬_D (∂Q/∂x - ∂P/∂y) dxdy
```

**记忆口诀**："后求后导，前求前导，后减前"

其中：
- L 取正向（逆时针）
- D 为 L 围成的区域
- ∂Q/∂x 对应 Q 项，∂P/∂y 对应 P 项

### 2.2 适用条件

1. **区域条件**：D 是平面单连通区域，边界 L 分段光滑
2. **函数条件**：P, Q 在 D 上有连续的一阶偏导数
3. **定向条件**：L 取正向（使 D 始终在左侧）

### 2.3 格林公式的应用

#### 应用1：计算曲线积分
将复杂的曲线积分转化为二重积分计算。

**例3**：计算 ∮_L x² dy - y² dx，其中 L 是 x² + y² = 4 的正向边界。

**解**：
```
P = -y², Q = x²
∂Q/∂x = 2x, ∂P/∂y = -2y

∮_L x² dy - y² dx = ∬_D (2x + 2y) dxdy
```

使用极坐标：x = r cos θ, y = r sin θ, 0 ≤ r ≤ 2, 0 ≤ θ ≤ 2π

```
= ∫_0^{2π} ∫_0^2 2r(cos θ + sin θ) r dr dθ
= 2 ∫_0^{2π} (cos θ + sin θ) dθ · ∫_0^2 r² dr
= 2 [sin θ - cos θ]_0^{2π} · [r³/3]_0^2
= 2 · 0 · 8/3 = 0
```

#### 应用2：计算平面区域的面积

取 P = 0, Q = x 或 P = -y, Q = 0，得：
```
A = ∮_L x dy = -∮_L y dx = (1/2) ∮_L (x dy - y dx)
```

**例4**：用曲线积分计算椭圆 x²/a² + y²/b² = 1 的面积。

**解**：
参数方程：x = a cos t, y = b sin t, 0 ≤ t ≤ 2π

```
A = (1/2) ∮_L (x dy - y dx)
  = (1/2) ∫_0^{2π} [a cos t · b cos t - b sin t · (-a sin t)] dt
  = (ab/2) ∫_0^{2π} dt = πab
```

### 2.4 路径无关性

若在单连通区域 D 内，对任意闭曲线 L 都有：
```
∮_L P dx + Q dy = 0
```

则曲线积分在 D 内与路径无关，只与起点终点有关。

#### 充要条件
在单连通区域 D 内，下列条件等价：

1. ∮_L P dx + Q dy = 0（对任意闭曲线 L）
2. ∫_L P dx + Q dy 与路径无关
3. ∂P/∂y = ∂Q/∂x 在 D 内恒成立
4. P dx + Q dy 是某个函数 u(x,y) 的全微分

#### 原函数的求法
若 ∂P/∂y = ∂Q/∂x，求 u(x,y) 使得 du = P dx + Q dy：

**方法一：不定积分法**
```
u(x,y) = ∫ P(x,y) dx + φ(y)
```
对 y 求偏导与 Q(x,y) 比较，确定 φ(y)。

**方法二：曲线积分法**
```
u(x,y) = ∫_{(x0,y0)}^{(x,y)} P dx + Q dy
```
选择方便的路径（如折线）计算。

### 2.5 考试重点与易错点

**高频考点**：
- 用格林公式计算曲线积分（★★★）
- 判断路径无关性（★★★）
- 计算平面图形面积（★★）

**易错点**：
1. 忘记检查格林公式的适用条件
2. 区域有"洞"时需要添加辅助线
3. 曲线方向搞反（正向是逆时针）
4. 路径无关条件只在单连通区域成立

## 3. 曲面积分

### 3.1 第一型曲面积分（对面积的曲面积分）

#### 概念
设 Σ 为光滑曲面，f(x,y,z) 在 Σ 上有界，则：
```
∬_Σ f(x,y,z) dS
```

#### 物理意义
- 曲面形构件的质量（f 为面密度）
- 曲面的质心、转动惯量

#### 计算方法

**方法一：显式方程 z = z(x,y)**

若曲面 Σ: z = z(x,y), (x,y) ∈ D_{xy}，则：
```
∬_Σ f(x,y,z) dS = ∬_{D_xy} f(x, y, z(x,y)) √[1 + (∂z/∂x)² + (∂z/∂y)²] dxdy
```

**面积元素**：
```
dS = √[1 + z_x² + z_y²] dxdy
```

**方法二：参数方程**

若 Σ: r(u,v) = (x(u,v), y(u,v), z(u,v)), (u,v) ∈ D，则：
```
dS = |r_u × r_v| dudv
```

#### 性质
- 与曲面的侧无关
- 与坐标系选择无关

#### 典型例题
**例5**：计算 ∬_Σ z dS,其中 Σ 是球面 x² + y² + z² = a² 的上半部分。

**解**：
使用球面坐标的参数方程：
```
x = a sin φ cos θ
y = a sin φ sin θ
z = a cos φ
```
其中 0 ≤ φ ≤ π/2, 0 ≤ θ ≤ 2π

面积元素：dS = a² sin φ dφ dθ

```
∬_Σ z dS = ∫_0^{2π} ∫_0^{π/2} (a cos φ) · a² sin φ dφ dθ
          = 2πa³ ∫_0^{π/2} sin φ cos φ dφ
          = 2πa³ · [sin²φ/2]_0^{π/2}
          = πa³
```

### 3.2 第二型曲面积分（对坐标的曲面积分）

#### 概念
设向量场 F = (P, Q, R)，有向曲面 Σ，则：
```
∬_Σ P dydz + Q dzdx + R dxdy
```

也可写成向量形式：
```
∬_Σ F·n dS = ∬_Σ F·dS
```
其中 n 是曲面的单位法向量。

#### 物理意义
- 流体通过曲面的通量（流量）
- 电场通过曲面的通量

#### 计算方法

**方法：投影法**

1. **对 dydz 的积分**：投影到 yOz 平面
   ```
   ∬_Σ P(x,y,z) dydz = ±∬_{D_yz} P(x(y,z), y, z) dydz
   ```
   - 曲面法向量与 x 轴正向成锐角取 "+"
   - 成钝角取 "-"

2. **对 dzdx 的积分**：投影到 zOx 平面
   ```
   ∬_Σ Q(x,y,z) dzdx = ±∬_{D_zx} Q(x, y(z,x), z) dzdx
   ```

3. **对 dxdy 的积分**：投影到 xOy 平面
   ```
   ∬_Σ R(x,y,z) dxdy = ±∬_{D_xy} R(x, y, z(x,y)) dxdy
   ```
   - 曲面法向量与 z 轴正向成锐角取 "+"
   - 成钝角取 "-"

**记忆技巧**：
- 对谁积分，就往哪个坐标平面投影
- 符号由法向量方向决定

#### 两类曲面积分的关系
```
∬_Σ P dydz + Q dzdx + R dxdy = ∬_Σ (P cos α + Q cos β + R cos γ) dS
```
其中 (cos α, cos β, cos γ) 是曲面外法向量的方向余弦。

#### 典型例题
**例6**：计算 ∬_Σ x dydz + y dzdx + z dxdy，其中 Σ 是球面 x² + y² + z² = a² 的外侧。

**解**：
由对称性，只需计算 z dxdy 部分（其他两项类似）。

球面上半部：z = √(a² - x² - y²), 法向量与 z 轴正向成锐角，取 "+"
球面下半部：z = -√(a² - x² - y²), 法向量与 z 轴正向成钝角,取 "-"

```
∬_Σ z dxdy = ∬_{上} z dxdy + ∬_{下} z dxdy
            = ∬_{D_xy} √(a² - x² - y²) dxdy - ∬_{D_xy} √(a² - x² - y²) dxdy
            = 0
```

同理，∬_Σ x dydz = 0, ∬_Σ y dzdx = 0

因此，原积分 = 0。

（注：也可直接用高斯公式更简便）

## 4. 高斯公式与斯托克斯公式

### 4.1 高斯公式（散度定理）

#### 公式表述
设空间闭区域 Ω 由分片光滑的闭曲面 Σ 围成，函数 P, Q, R 在 Ω 上具有一阶连续偏导数,则：

```
∬_Σ P dydz + Q dzdx + R dxdy = ∭_Ω (∂P/∂x + ∂Q/∂y + ∂R/∂z) dV
```

其中 Σ 取外侧（向外为正）。

**向量形式**：
```
∬_Σ F·n dS = ∭_Ω div F dV = ∭_Ω ∇·F dV
```

#### 散度的定义
```
div F = ∇·F = ∂P/∂x + ∂Q/∂y + ∂R/∂z
```

#### 物理意义
- 左侧：通过闭曲面的总通量
- 右侧：区域内所有源的强度之和
- 散度：单位体积内的通量源密度

#### 记忆技巧
"P 对 x, Q 对 y, R 对 z，全部相加"

#### 典型例题
**例7**：用高斯公式计算 ∬_Σ x³ dydz + y³ dzdx + z³ dxdy，其中 Σ 是球面 x² + y² + z² = a² 的外侧。

**解**：
```
P = x³, Q = y³, R = z³
∂P/∂x = 3x², ∂Q/∂y = 3y², ∂R/∂z = 3z²

∬_Σ = ∭_Ω 3(x² + y² + z²) dV
```

使用球坐标：x² + y² + z² = r²
```
= 3 ∫_0^{2π} ∫_0^π ∫_0^a r² · r² sin φ dr dφ dθ
= 3 · 2π · 2 · ∫_0^a r⁴ dr
= 12π · [r⁵/5]_0^a
= 12πa⁵/5
```

---

### 4.2 斯托克斯公式（旋度定理）

#### 公式表述
设 Σ 为分片光滑的有向曲面,其边界为分段光滑的有向闭曲线 L（L 的正向与 Σ 的侧符合右手法则），函数 P, Q, R 在 Σ 上具有一阶连续偏导数，则：

```
∮_L P dx + Q dy + R dz = ∬_Σ (∂R/∂y - ∂Q/∂z) dydz + (∂P/∂z - ∂R/∂x) dzdx + (∂Q/∂x - ∂P/∂y) dxdy
```

**向量形式**：
```
∮_L F·dr = ∬_Σ (∇×F)·n dS = ∬_Σ curl F·n dS
```

#### 旋度的定义
```
curl F = ∇×F = | i    j    k   |
                | ∂/∂x ∂/∂y ∂/∂z |
                | P    Q    R   |

= (∂R/∂y - ∂Q/∂z) i + (∂P/∂z - ∂R/∂x) j + (∂Q/∂x - ∂P/∂y) k
```

#### 记忆技巧
用行列式展开：
```
     | i    j    k   |
∇×F = | ∂_x  ∂_y  ∂_z |
     | P    Q    R   |
```

#### 物理意义
- 旋度：描述向量场的旋转强度
- 环流量：流体沿闭曲线的循环流动

#### 典型例题
**例8**：用斯托克斯公式计算 ∮_L y dx + z dy + x dz，其中 L 是平面 x + y + z = 1 与三个坐标面的交线，沿正向（从 z 轴正向看去为逆时针）。

**解**：
```
P = y, Q = z, R = x

∂R/∂y - ∂Q/∂z = 0 - 1 = -1
∂P/∂z - ∂R/∂x = 0 - 1 = -1
∂Q/∂x - ∂P/∂y = 0 - 1 = -1

∮_L = ∬_Σ (-dydz - dzdx - dxdy)
```

Σ 是三角形 x + y + z = 1 (x,y,z ≥ 0) 在第一卦限部分，法向量为 (1,1,1)/√3。

投影到 xOy 平面：z = 1 - x - y
```
dS = √3 dxdy
法向量：n = (1,1,1)/√3

∮_L = ∬_Σ (-1,-1,-1)·(1,1,1)/√3 · √3 dxdy
    = -3 ∬_{D_xy} dxdy
    = -3 · (三角形面积)
    = -3 · (1/2 · 1 · 1)
    = -3/2
```

## 5. 向量场论

### 5.1 哈密顿算子 ∇

#### 定义
```
∇ = (∂/∂x, ∂/∂y, ∂/∂z) = i ∂/∂x + j ∂/∂y + k ∂/∂z
```

∇ 是一个向量微分算子，可以作用于标量场和向量场。

### 5.2 梯度（Gradient）

#### 定义
标量场 u(x,y,z) 的梯度：
```
grad u = ∇u = (∂u/∂x, ∂u/∂y, ∂u/∂z)
```

#### 性质
1. **方向**：指向 u 增长最快的方向
2. **大小**：|∇u| = 最大方向导数
3. **梯度垂直于等值面**：∇u ⊥ 等值面 u = C

#### 几何意义
- 梯度方向：函数值增长最快的方向
- 梯度模：最大变化率
- 梯度垂直于等高线/等值面

#### 计算公式
平面：grad u = (∂u/∂x, ∂u/∂y)
空间：grad u = (∂u/∂x, ∂u/∂y, ∂u/∂z)

---

### 5.3 散度（Divergence）

#### 定义
向量场 F = (P, Q, R) 的散度：
```
div F = ∇·F = ∂P/∂x + ∂Q/∂y + ∂R/∂z
```

#### 物理意义
- 描述场的通量密度（源的强度）
- div F > 0：源（发散点）
- div F < 0：汇（会聚点）
- div F = 0：无源场

#### 高斯公式的微分形式
```
∬_Σ F·n dS = ∭_Ω div F dV
```

---

### 5.4 旋度（Curl）

#### 定义
向量场 F = (P, Q, R) 的旋度：
```
curl F = ∇×F = | i    j    k   |
                | ∂_x  ∂_y  ∂_z |
                | P    Q    R   |
```

展开：
```
curl F = (∂R/∂y - ∂Q/∂z) i + (∂P/∂z - ∂R/∂x) j + (∂Q/∂x - ∂P/∂y) k
```

#### 物理意义
- 描述场的旋转强度
- curl F = 0：无旋场（保守场）
- |curl F|：单位面积上的环流强度

#### 斯托克斯公式的微分形式
```
∮_L F·dr = ∬_Σ (curl F)·n dS
```

---

### 5.5 重要公式与恒等式

#### 二阶微分算子

1. **拉普拉斯算子**：
   ```
   Δu = ∇²u = div(grad u) = ∂²u/∂x² + ∂²u/∂y² + ∂²u/∂z²
   ```

2. **无旋场的充要条件**：
   ```
   curl(grad u) = ∇×(∇u) = 0
   ```

3. **无散场**：
   ```
   div(curl F) = ∇·(∇×F) = 0
   ```

#### 运算法则

1. **梯度**：
   - ∇(u + v) = ∇u + ∇v
   - ∇(uv) = u∇v + v∇u
   - ∇(u/v) = (v∇u - u∇v)/v²

2. **散度**：
   - div(F + G) = div F + div G
   - div(φF) = φ div F + F·∇φ

3. **旋度**：
   - curl(F + G) = curl F + curl G
   - curl(φF) = φ curl F + ∇φ × F

---

### 5.6 保守场与势函数

#### 保守场的充要条件
在单连通区域内，下列条件等价：

1. F 是保守场（F = ∇φ）
2. curl F = 0（无旋场）
3. ∮_L F·dr = 0（沿任意闭曲线环流为零）
4. ∫_L F·dr 与路径无关

#### 势函数的求法
若 curl F = 0，求 φ 使得 F = ∇φ：

```
∂φ/∂x = P
∂φ/∂y = Q
∂φ/∂z = R
```

**方法**：
```
φ(x,y,z) = ∫_{(x0,y0,z0)}^{(x,y,z)} P dx + Q dy + R dz
```

### 5.7 典型例题

**例9**：验证向量场 F = (y, x, 0) 是否为保守场？若是，求其势函数。

**解**：
计算旋度：
```
curl F = | i   j   k  |
         | ∂_x ∂_y ∂_z|
         | y   x   0  |

= (0-0)i - (0-0)j + (1-1)k = 0
```

curl F = 0，故 F 是保守场。

求势函数 φ：
```
∂φ/∂x = y  →  φ = xy + f(y,z)
∂φ/∂y = x  →  ∂f/∂y = 0  →  f = g(z)
∂φ/∂z = 0  →  g'(z) = 0  →  g = C

φ(x,y,z) = xy + C
```

## 6. 级数理论深化

### 6.1 正项级数审敛法总结

#### 1. 比较判别法
若 0 ≤ u_n ≤ v_n：
- Σv_n 收敛 → Σu_n 收敛
- Σu_n 发散 → Σv_n 发散

**常用比较级数**：
- 几何级数：Σar^n（|r|<1 收敛）
- p-级数：Σ1/n^p（p>1 收敛）

#### 2. 比较判别法的极限形式
```
lim (u_n / v_n) = l (n→∞)
```

- 0 < l < ∞：Σu_n 与 Σv_n 同敛散
- l = 0，Σv_n 收敛 → Σu_n 收敛
- l = ∞，Σv_n 发散 → Σu_n 发散

#### 3. 比值判别法（达朗贝尔判别法）
```
lim (u_{n+1} / u_n) = ρ (n→∞)
```

- ρ < 1：级数收敛
- ρ > 1：级数发散
- ρ = 1：不确定

**适用**：含阶乘、指数项

#### 4. 根值判别法（柯西判别法）
```
lim ⁿ√u_n = ρ (n→∞)
```

- ρ < 1：级数收敛
- ρ > 1：级数发散
- ρ = 1：不确定

**适用**：含 n 次方项

#### 5. 积分判别法
若 f(x) 在 [1,∞) 上单调递减且非负，则：
```
Σf(n) 与 ∫_1^∞ f(x) dx 同敛散
```

**例**：p-级数 Σ1/n^p
```
∫_1^∞ 1/x^p dx = { [x^{1-p}/(1-p)]_1^∞  p≠1
                  { [ln x]_1^∞            p=1

p > 1 收敛，p ≤ 1 发散
```

---

### 6.2 交错级数

#### 莱布尼茨判别法
若交错级数 Σ(-1)^{n-1} u_n 满足：
1. u_n ≥ u_{n+1}（单调递减）
2. lim u_n = 0

则级数收敛，且余项估计：|R_n| ≤ u_{n+1}

**典型例子**：
```
Σ(-1)^{n-1}/n = 1 - 1/2 + 1/3 - 1/4 + ... = ln 2
```

---

### 6.3 绝对收敛与条件收敛

#### 定义
- **绝对收敛**：Σ|u_n| 收敛
- **条件收敛**：Σu_n 收敛但 Σ|u_n| 发散

#### 重要定理
1. 绝对收敛 → 收敛（但反之不一定）
2. 绝对收敛级数的项可以任意重排，和不变
3. 条件收敛级数重排后和可能改变

#### 审敛步骤
1. 先判断 Σ|u_n| 是否收敛（绝对收敛判别）
2. 若 Σ|u_n| 发散，再判断 Σu_n 是否收敛（条件收敛判别）

---

### 6.4 幂级数

#### 收敛半径与收敛域

**标准形式**：Σa_n x^n

**收敛半径 R 的求法**：

1. **比值法**：
   ```
   R = lim |a_n / a_{n+1}| (n→∞)
   ```

2. **根值法**：
   ```
   R = 1 / lim ⁿ√|a_n| (n→∞)
   ```

**收敛域**：
- 在 |x| < R 内绝对收敛
- 在 |x| > R 外发散
- 端点 x = ±R 需单独讨论

#### 幂级数的性质

1. **逐项求导**：
   ```
   (Σa_n x^n)' = Σn a_n x^{n-1}  （收敛半径不变）
   ```

2. **逐项积分**：
   ```
   ∫Σa_n x^n dx = Σa_n x^{n+1}/(n+1) + C  （收敛半径不变）
   ```

#### 常用幂级数展开

```
e^x = Σ x^n / n! = 1 + x + x²/2! + x³/3! + ...  (|x| < ∞)

sin x = Σ (-1)^n x^{2n+1} / (2n+1)! = x - x³/3! + x⁵/5! - ...  (|x| < ∞)

cos x = Σ (-1)^n x^{2n} / (2n)! = 1 - x²/2! + x⁴/4! - ...  (|x| < ∞)

ln(1+x) = Σ (-1)^{n-1} x^n / n = x - x²/2 + x³/3 - ...  (|x| ≤ 1, x≠-1)

1/(1-x) = Σ x^n = 1 + x + x² + x³ + ...  (|x| < 1)
```

## 7. 傅里叶级数

### 7.1 傅里叶级数的定义

#### 三角级数的一般形式
```
f(x) = a₀/2 + Σ(aₙ cos nx + bₙ sin nx)  (n=1,2,3,...)
```

#### 傅里叶系数
设 f(x) 是周期为 2π 的函数，则：

```
a₀ = (1/π) ∫_{-π}^π f(x) dx

aₙ = (1/π) ∫_{-π}^π f(x) cos nx dx  (n=1,2,3,...)

bₙ = (1/π) ∫_{-π}^π f(x) sin nx dx  (n=1,2,3,...)
```

**记忆口诀**：
- a₀：平均值的 2 倍
- aₙ：f(x) 与 cos nx 的内积
- bₙ：f(x) 与 sin nx 的内积

---

### 7.2 狄利克雷收敛定理

#### 收敛条件
若 f(x) 满足：
1. 在一个周期内连续或只有有限个第一类间断点
2. 在一个周期内只有有限个极值点

则 f(x) 的傅里叶级数收敛，且：
- 在连续点 x 处：级数收敛于 f(x)
- 在间断点 x 处：级数收敛于 [f(x⁺) + f(x⁻)]/2

#### 吉布斯现象
在间断点附近，傅里叶级数会出现约 9% 的过冲现象。

---

### 7.3 奇延拓与偶延拓

#### 偶函数的傅里叶级数（余弦级数）
若 f(x) 是偶函数，则 bₙ = 0，只有余弦项：
```
f(x) = a₀/2 + Σ aₙ cos nx

aₙ = (2/π) ∫_0^π f(x) cos nx dx  (n=0,1,2,...)
```

#### 奇函数的傅里叶级数（正弦级数）
若 f(x) 是奇函数，则 aₙ = 0，只有正弦项：
```
f(x) = Σ bₙ sin nx

bₙ = (2/π) ∫_0^π f(x) sin nx dx  (n=1,2,3,...)
```

#### 延拓技巧
对定义在 [0, π] 上的函数 f(x)：
- **偶延拓**：f(-x) = f(x)，展开为余弦级数
- **奇延拓**：f(-x) = -f(x)，展开为正弦级数

选择延拓方式的原则：
- 若边界条件 f(0) = f(π) = 0，选奇延拓
- 若需要 f'(0) = f'(π) = 0，选偶延拓

---

### 7.4 周期为 2L 的函数的傅里叶级数

#### 公式
```
f(x) = a₀/2 + Σ(aₙ cos(nπx/L) + bₙ sin(nπx/L))

a₀ = (1/L) ∫_{-L}^L f(x) dx

aₙ = (1/L) ∫_{-L}^L f(x) cos(nπx/L) dx

bₙ = (1/L) ∫_{-L}^L f(x) sin(nπx/L) dx
```

#### 换元法
令 t = πx/L，将周期 2L 变换为周期 2π。

---

### 7.5 典型例题

**例10**：将 f(x) = x 在 (-π, π) 上展开为傅里叶级数。

**解**：
f(x) = x 是奇函数，aₙ = 0。

```
bₙ = (1/π) ∫_{-π}^π x sin nx dx
   = (2/π) ∫_0^π x sin nx dx  （利用对称性）
```

分部积分：
```
= (2/π) [-x cos nx / n]_0^π + (2/π) ∫_0^π (cos nx / n) dx
= (2/π) [-π cos nπ / n + 0] + (2/π) [sin nx / n²]_0^π
= -2 cos nπ / n
= 2(-1)^{n+1} / n
```

因此：
```
x = 2 Σ [(-1)^{n+1} / n] sin nx
  = 2(sin x - sin 2x/2 + sin 3x/3 - ...)
```

---

**例11**：将 f(x) = |x| 在 (-π, π) 上展开为傅里叶级数。

**解**：
f(x) = |x| 是偶函数，bₙ = 0。

```
a₀ = (1/π) ∫_{-π}^π |x| dx = (2/π) ∫_0^π x dx = π

aₙ = (1/π) ∫_{-π}^π |x| cos nx dx
   = (2/π) ∫_0^π x cos nx dx
```

分部积分：
```
= (2/π) [x sin nx / n]_0^π - (2/π) ∫_0^π (sin nx / n) dx
= 0 + (2/π) [cos nx / n²]_0^π
= (2/πn²) [cos nπ - 1]
= (2/πn²) [(-1)^n - 1]
```

当 n 为偶数：aₙ = 0
当 n 为奇数：aₙ = -4/(πn²)

因此：
```
|x| = π/2 - (4/π) Σ [cos(2k-1)x / (2k-1)²]  (k=1,2,3,...)
    = π/2 - (4/π)(cos x + cos 3x/9 + cos 5x/25 + ...)
```

### 7.6 帕塞瓦尔等式

若 f(x) 在 (-π, π) 上平方可积，则：
```
(1/π) ∫_{-π}^π f²(x) dx = a₀²/2 + Σ(aₙ² + bₙ²)
```

**意义**：能量守恒（时域能量 = 频域能量）
