# 数学基础学习资料

本目录包含了高等数学、线性代数和概率论的学习文档和Python代码示例。

## 文件结构

```
Math_Foundations/
├── README.md                         # 本文件
├── calculus/                         # 高等数学
│   ├── calculus_guide.md             # 学习指南
│   └── calculus_examples.py          # 代码示例
├── linear_algebra/                   # 线性代数
│   ├── linear_algebra_guide.md       # 学习指南
│   └── linear_algebra_examples.py    # 代码示例
└── probability/                      # 概率论
```

## 环境要求

确保已安装以下Python库：

```bash
pip install numpy matplotlib scipy sympy seaborn
```

## 使用方法

### 1. 高等数学

**理论学习**：阅读 `calculus/calculus_guide.md`
**代码实践**：运行 `calculus/calculus_examples.py`

```bash
cd calculus
python calculus_examples.py
```

包含内容：
- 极限与连续的概念和计算
- 导数与微分的可视化
- 积分的计算和几何意义
- 级数展开和收敛性
- 多元函数的偏导数和梯度
- 常微分方程求解

### 2. 线性代数

**理论学习**：阅读 `linear_algebra/linear_algebra_guide.md`
**代码实践**：运行 `linear_algebra/linear_algebra_examples.py`

```bash
cd linear_algebra
python linear_algebra_examples.py
```

包含内容：
- 矩阵运算和向量操作
- 线性方程组求解
- 向量空间和基变换
- 特征值和特征向量
- 矩阵分解（LU、QR、SVD）
- 线性变换的可视化

### 3. 概率论

**理论学习**：阅读 `probability/probability_guide.md`
**代码实践**：运行 `probability/probability_examples.py`

```bash
cd probability
python probability_examples.py
```

包含内容：
- 基础概率概念和计算
- 条件概率和贝叶斯定理
- 离散和连续概率分布
- 大数定律和中心极限定理
- 假设检验
- 蒙特卡罗模拟

## 学习建议

### 循序渐进
1. 先阅读理论文档，理解基本概念
2. 运行代码示例，观察可视化效果
3. 修改代码参数，探索结果变化
4. 尝试解决课后练习题

### 动手实践
- 修改函数参数，观察图形变化
- 尝试不同的矩阵和向量
- 改变概率分布的参数
- 设计自己的实验

### 理论联系实际
- 思考每个概念的几何意义
- 理解公式背后的直观含义
- 将抽象概念与具体例子结合
- 注意不同数学分支之间的联系
