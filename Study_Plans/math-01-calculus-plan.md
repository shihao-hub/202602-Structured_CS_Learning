## 学习阶段规划

### 基础阶段（2月下旬 - 5月底）

**目标**：建立完整知识体系，理解所有基础概念和定理

#### 第1-2周：函数、极限、连续

| 日期 | 内容 | 学习要点 |
|------|------|----------|
| Week1 | 函数概念与性质 | 基本初等函数、复合函数、隐函数、参数方程 |
| Week1 | 极限概念 | 数列极限、函数极限、ε-δ语言（了解） |
| Week2 | 极限计算 | 等价无穷小、洛必达法则、泰勒展开 |
| Week2 | 连续性 | 间断点分类、连续函数性质（零点定理、介值定理） |

**重点题型**：
- [ ] 极限计算（0/0、∞/∞型）
- [ ] 等价无穷小替换
- [ ] 泰勒公式应用
- [ ] 判断间断点类型

#### 第3-4周：一元函数微分学

| 日期 | 内容 | 学习要点 |
|------|------|----------|
| Week3 | 导数定义与计算 | 导数定义、求导法则、高阶导数 |
| Week3 | 微分与微分中值定理 | 微分概念、罗尔定理、拉格朗日、柯西 |
| Week4 | 导数应用 | 单调性、极值、凹凸性、拐点 |
| Week4 | 综合应用 | 不等式证明、曲线渐近线、曲率 |

**重点题型**：
- [ ] 导数定义求导
- [ ] 中值定理证明题
- [ ] 单调性与极值判断
- [ ] 不等式证明

#### 第5-6周：一元函数积分学

| 日期 | 内容 | 学习要点 |
|------|------|----------|
| Week5 | 不定积分 | 基本积分公式、换元法、分部积分 |
| Week5 | 有理函数积分 | 部分分式、三角函数有理化 |
| Week6 | 定积分 | 定积分定义、牛顿-莱布尼茨公式 |
| Week6 | 定积分应用 | 面积、体积、弧长、物理应用 |

**重点题型**：
- [ ] 换元积分（三角换元、根式换元）
- [ ] 分部积分
- [ ] 定积分计算
- [ ] 变上限积分求导

#### 第7周：向量代数与空间解析几何

| 日期 | 内容 | 学习要点 |
|------|------|----------|
| Week7 | 向量运算 | 点积、叉积、混合积 |
| Week7 | 空间曲面与曲线 | 平面方程、直线方程、二次曲面 |

**重点题型**：
- [ ] 向量运算
- [ ] 平面与直线的位置关系
- [ ] 点到平面/直线距离

#### 第8-9周：多元函数微分学

| 日期 | 内容 | 学习要点 |
|------|------|----------|
| Week8 | 多元函数基础 | 极限、连续、偏导数、全微分 |
| Week8 | 复合函数与隐函数 | 链式法则、隐函数存在定理 |
| Week9 | 极值与最值 | 无条件极值、条件极值（拉格朗日乘数法） |
| Week9 | 方向导数与梯度 | 方向导数计算、梯度概念 |

**重点题型**：
- [ ] 复合函数偏导数
- [ ] 隐函数偏导数
- [ ] 条件极值（拉格朗日乘数法）
- [ ] 梯度与方向导数

#### 第10-11周：多元函数积分学

| 日期 | 内容 | 学习要点 |
|------|------|----------|
| Week10 | 二重积分 | 计算方法（直角坐标、极坐标）、交换积分次序 |
| Week10 | 三重积分 | 计算方法（直角、柱面、球面坐标） |
| Week11 | 曲线积分 | 第一型、第二型曲线积分、格林公式 |
| Week11 | 曲面积分 | 第一型、第二型曲面积分、高斯公式、斯托克斯公式 |

**重点题型**：
- [ ] 二重积分计算（极坐标变换）
- [ ] 三重积分计算
- [ ] 格林公式应用
- [ ] 高斯公式、斯托克斯公式

#### 第12-13周：无穷级数

| 日期 | 内容 | 学习要点 |
|------|------|----------|
| Week12 | 数项级数 | 敛散性判别、绝对收敛与条件收敛 |
| Week12 | 幂级数 | 收敛域、和函数 |
| Week13 | 傅里叶级数 | 傅里叶系数、收敛定理 |

**重点题型**：
- [ ] 级数敛散性判断
- [ ] 幂级数收敛域
- [ ] 幂级数求和
- [ ] 傅里叶系数计算

#### 第14周：常微分方程

| 日期 | 内容 | 学习要点 |
|------|------|----------|
| Week14 | 一阶方程 | 可分离变量、齐次方程、一阶线性 |
| Week14 | 高阶方程 | 可降阶方程、二阶线性常系数 |

**重点题型**：
- [ ] 一阶线性微分方程
- [ ] 二阶常系数齐次/非齐次
- [ ] 微分方程应用题

### 强化阶段（6月 - 9月）

**目标**：专题突破，提高解题速度和准确率

#### 每周安排模板

| 时间 | 内容 |
|------|------|
| 周一-周三 | 专题训练（每天1-2个专题） |
| 周四-周五 | 综合练习 |
| 周六 | 错题回顾 |
| 周日 | 模拟测试 |

#### 专题列表

1. **极限专题**
   - 0/0型、∞/∞型极限
   - 1^∞型极限
   - 数列极限

2. **中值定理专题**
   - 罗尔定理应用
   - 拉格朗日中值定理应用
   - 柯西中值定理应用
   - 泰勒公式应用

3. **积分专题**
   - 三角函数积分
   - 有理函数积分
   - 定积分几何应用

4. **多元微分专题**
   - 复合函数求导
   - 隐函数求导
   - 极值与最值

5. **多元积分专题**
   - 二重积分换序
   - 极坐标与柱面坐标
   - 三大公式（格林、高斯、斯托克斯）

6. **级数专题**
   - 敛散性判断
   - 幂级数展开
   - 傅里叶级数

### 冲刺阶段（10月 - 12月）

**目标**：真题训练，查漏补缺，稳定发挥

#### 真题训练计划

| 时间 | 内容 |
|------|------|
| 10月 | 近15年真题（2011-2025），每周2-3套 |
| 11月 | 真题二刷（错题为主）+ 模拟卷 |
| 12月 | 重点公式回顾 + 模拟考试 |

#### 真题使用方法

1. **限时训练**：3小时内完成
2. **详细批改**：标记每道题的知识点
3. **错题分析**：记录错误原因
4. **同类归纳**：相似题型归类

## 学习方法建议

### 薄弱基础提升策略

1. **降低速度**：基础薄弱不要急，每个概念都要理解透
2. **多做基础题**：先做简单题建立信心
3. **及时复习**：新知识24小时内必须复习
4. **建立错题本**：分章节整理错题

### 高效学习技巧

```
每日学习流程：
1. 复习昨日内容（15分钟）
2. 学习新知识（60分钟）
3. 做配套练习（45分钟）
4. 总结归纳（15分钟）
```

### 公式记忆方法

1. **理解推导**：不要死记，理解公式来源
2. **高频使用**：每天默写重要公式
3. **关联记忆**：找公式之间的联系
4. **错题强化**：在错题中巩固公式
