# 数学三科学习路线图

> 高等数学、线性代数、概率论的学习顺序与时间安排

## 整体顺序

```
高等数学（先学，基础）
    ↓
线性代数（高数学到一半时穿插）
    ↓
概率论（需要高数积分基础）
```

**原因**：概率论需要大量积分计算，线代相对独立但也需要一些高数基础。

## 时间线总览（基础阶段：2月下旬 - 5月底）

| 时间 | 高等数学 | 线性代数 | 概率论 |
|------|----------|----------|--------|
| 2月下旬-3月初 | 函数、极限、导数 | - | - |
| 3月 | 积分、向量 | **开始** | - |
| 4月上旬 | 多元微分 | 进行中 | - |
| 4月中旬 | 多元积分 | **完成基础** | **开始** |
| 5月 | 级数、微分方程 | 强化 | 进行中 |
| 5月底 | **完成基础** | - | **完成基础** |

## 具体学习路线

### 阶段一：纯高数（2月下旬 - 3月初，约2周）

**每周18小时全给高数**

| 周 | 内容 |
|----|------|
| 第1周 | 函数、极限、连续 |
| 第2周 | 导数、微分、中值定理 |

**为什么先纯学高数**：极限和导数是后面所有内容的基础，必须打牢。

### 阶段二：高数为主 + 线代穿插（3月 - 4月中旬，约6周）

**每周分配**：高数12小时 + 线代6小时

| 周 | 高数内容 | 线代内容 |
|----|----------|----------|
| 第3周 | 导数应用 | 行列式 |
| 第4周 | 不定积分 | 矩阵（上） |
| 第5周 | 定积分 | 矩阵（下） |
| 第6周 | 向量与空间几何 | 向量组 |
| 第7周 | 多元函数微分（上） | 线性方程组 |
| 第8周 | 多元函数微分（下） | 特征值、二次型 |

**为什么穿插线代**：
- 高数和线代交替学习，减少疲劳
- 线代7周内容正好在这个阶段完成

### 阶段三：高数 + 概率并行（4月中旬 - 5月底，约6周）

**每周分配**：高数10小时 + 概率8小时

| 周 | 高数内容 | 概率内容 |
|----|----------|----------|
| 第9周 | 二重积分 | 随机事件与概率 |
| 第10周 | 三重积分 | 一维随机变量 |
| 第11周 | 曲线积分 | 二维随机变量（上） |
| 第12周 | 曲面积分 | 二维随机变量（下） |
| 第13周 | 无穷级数 | 数字特征、大数定律 |
| 第14周 | 常微分方程 | 数理统计、参数估计 |

**为什么这时开始概率**：
- 概率需要积分计算，此时高数积分已学完
- 二维随机变量要用到二重积分

## 每日学习建议

### 工作日（4.5小时）

```
阶段一（纯高数）：
21:00-23:30  高数视频+例题（2.5h）
07:30-09:30  高数课后题（2h）

阶段二/三（并行）：
21:00-22:30  高数视频+例题（1.5h）
22:30-23:30  线代/概率视频+例题（1h）
07:30-09:30  做题（高数1h + 线代/概率1h）
```

### 周末（10.5小时）

```
上午 08:30-12:00  高数（3.5h）
下午 14:00-17:00  线代/概率（3h）
晚上 19:30-22:30  做题+错题回顾（3h）
机动 1h          补进度或休息
```

## 关键检查点

| 时间点 | 应完成内容 | 自检方法 |
|--------|------------|----------|
| 3月初 | 高数：极限+导数 | 能熟练计算各类极限、求导 |
| 4月初 | 高数：积分 / 线代：矩阵 | 能做换元积分、求逆矩阵 |
| 4月底 | 高数：多元微分 / 线代：全部 | 能做偏导、对角化 |
| 5月底 | 高数+概率：全部基础 | 能做二重积分、参数估计 |

## 学习方法提醒

> 详见 [math-study-methodology.md](math-study-methodology.md)

核心要点：
1. **小单元学习**：每学完一小节（30-60min）立刻做2-3道题
2. **视频为主**：基础阶段视频是老师，书本是字典
3. **例题当题做**：看例题时遮住答案先想
4. **翻书确认公式**：视频后快速翻书确认公式（30秒）

## 一句话总结

> 2-3月主攻高数 → 3-4月高数+线代 → 4-5月高数+概率
