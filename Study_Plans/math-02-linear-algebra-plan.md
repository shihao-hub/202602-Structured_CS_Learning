## 重要公式速查

### 行列式

| 公式 | 说明 |
|------|------|
| \|AB\| = \|A\|\|B\| | 乘积行列式 |
| \|A^T\| = \|A\| | 转置行列式 |
| \|kA\| = k^n\|A\| | 数乘行列式 |
| \|A^{-1}\| = 1/\|A\| | 逆矩阵行列式 |
| \|A*\| = \|A\|^{n-1} | 伴随矩阵行列式 |

### 矩阵

| 公式 | 说明 |
|------|------|
| (AB)^T = B^T A^T | 乘积转置 |
| (AB)^{-1} = B^{-1}A^{-1} | 乘积逆 |
| A^{-1} = A*/\|A\| | 伴随矩阵求逆 |
| r(AB) ≤ min{r(A), r(B)} | 秩不等式 |
| r(A+B) ≤ r(A) + r(B) | 秩不等式 |

### 特征值

| 公式 | 说明 |
|------|------|
| Σλ_i = tr(A) | 特征值之和 = 迹 |
| Πλ_i = \|A\| | 特征值之积 = 行列式 |
| λ(A^k) = λ^k | 矩阵幂特征值 |
| λ(A^{-1}) = 1/λ | 逆矩阵特征值 |
| λ(A*) = \|A\|/λ | 伴随矩阵特征值 |

## 学习方法建议

### 线代学习特点

1. **概念抽象**：需要大量举例理解
2. **环环相扣**：前面不懂后面更难
3. **计算量大**：细心是关键
4. **技巧性强**：多总结解题套路

### 高效学习策略

1. **建立知识网络**：理解各章联系
2. **掌握基本变换**：初等变换是核心工具
3. **记忆重要结论**：节省计算时间
4. **多做综合题**：培养整体思维

### 易错点提醒

- 矩阵乘法不满足交换律
- |kA| = k^n|A|，不是 k|A|
- 相似矩阵≠相等矩阵
- 特征向量线性无关≠正交
- 正定矩阵必须是实对称矩阵
