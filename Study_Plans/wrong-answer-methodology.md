# 错题处理方法论

> 适用于408选择题、大题，也适用于数学和英语

## 核心认知

```
做错 = 发现了不会的知识点 = 进步的机会
做对 = 什么也没学到
```

基础阶段错很多是正常的，不要有挫败感。

## 错题三步法

每道错题都做这三步：

```
第1步：分类错因（为什么错？）
第2步：补知识点（缺什么补什么）
第3步：标记等级（方便复习）
```

### 第1步：分类错因

| 错因类型 | 表现 | 处理方式 |
|----------|------|----------|
| **概念不清** | 根本不知道这个知识点 | 回去看视频/翻书，重新理解 |
| **理解偏差** | 知道但理解错了 | 在题目旁写正确理解 |
| **审题失误** | 会做但看错了条件 | 标记"粗心"，提醒自己 |
| **方法不熟** | 知道用什么但不会算 | 多练同类型题 |

**最重要的**：在错题旁边用一句话写下"为什么错"。

### 第2步：补知识点

```
例：排序题做错了
    ↓
错因：分不清快排和归并的稳定性
    ↓
翻书确认：快排不稳定，归并稳定
    ↓
在题目旁写：「快排不稳定！」
```

不需要专门的错题本，**直接在王道书上标记**就行：
- 错题画个圈
- 旁边写错因（一句话）

### 第3步：标记等级

在错题旁标记：

| 标记 | 含义 | 复习频率 |
|------|------|----------|
| `x` | 完全不会 | 必须再做一遍 |
| `△` | 模糊/半懂 | 强化阶段回顾 |
| `○` | 粗心做错 | 提醒即可 |

## 如何降低重复犯错

```
第一次做错：标记 + 写错因
     ↓（强化阶段）
第二次遇到：只刷标记的错题
     ↓
还是错：说明知识点有漏洞，重新翻书
     ↓
做对了：降低标记等级
```

**核心就一句话**：错题旁边写一句话"为什么错"，下次复习时先看这句话再做题。

## 不同阶段的错题处理

### 基础阶段（选择题）

- 错了正常，不要纠结正确率
- 重点是标记错题 + 写错因
- 不需要立刻重做，留给强化阶段

### 强化阶段（大题）

- 大题不会写也正常，先看答案学思路
- 记录解题步骤和关键公式
- 同类型题做2-3道巩固

### 真题阶段

- 统计各章节错题数量，定位薄弱章节
- 回到王道书对应章节补强
