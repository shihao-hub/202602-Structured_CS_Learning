# 英语一学习计划

> 英语一是考研中分值高且拉分明显的科目，超级薄弱需要持续投入

## 考试题型与分值分布

| 题型 | 题数 | 分值 | 难度 | 目标得分 |
|------|------|------|------|----------|
| 完形填空 | 20题 | 10分 | 较难 | 6-7分 |
| 阅读理解A | 20题 | 40分 | 难 | 28-32分 |
| 阅读理解B(新题型) | 5题 | 10分 | 中等 | 6-8分 |
| 翻译 | 5题 | 10分 | 较难 | 6-7分 |
| 小作文 | 1题 | 10分 | 中等 | 7-8分 |
| 大作文 | 1题 | 20分 | 中等 | 12-14分 |
| **总计** | - | **100分** | - | **65-76分** |

## 目标分析

当前水平：超级薄弱
目标分数：70分

**分数分配策略**：
- 阅读理解（50分）：目标34-40分（阅读是大头，必须重点突破）
- 写作（30分）：目标19-22分（可以通过模板提分）
- 完形+翻译（20分）：目标12-14分（难度大，保底即可）

#### 快速过词方案（推荐）

> 适合基础薄弱、时间紧张的考生，目标是先快速过一遍全部词汇，后续再巩固

**方案对比**：

| 方案 | 每日新词 | 完成时间 | 每日时长 |
|------|----------|----------|----------|
| 激进方案 | 150词 | 37天（约5周） | 75-90分钟 |
| **推荐方案** | 100词 | 55天（约8周） | 50-60分钟 |
| 保守方案 | 80词 | 69天（约10周） | 40-50分钟 |

**快速过词策略**：

目标：认识即可，不求拼写

```
每个单词只需：
- 看英文 → 知道中文意思
- 每词用时：20-30秒
- 100词 × 30秒 = 50分钟
```

**三轮学习法**：

```
第一轮（快速过）：2月底 - 4月底
├── 每天100词新词（50分钟）
├── 复习昨天的词（20分钟）
└── 总时长：约70分钟/天

第二轮（巩固）：5月 - 7月
├── 每天复习200-300词
├── 重点攻克一轮不熟的词
└── 配合阅读真题巩固

第三轮（强化）：8月 - 12月
├── 结合真题语境复习
├── 只背不熟的词
└── 维持状态
```

**时间线**：

```
2月23日 开始
4月20日 一轮完成（约55天）
7月底   二轮完成
12月    三轮巩固完成
```

**快速过词技巧**：

1. **只看中文核心释义**，不纠结多个义项
2. **不背拼写**，能认就行
3. **用APP刷**，比看书快
4. **标记不熟的词**，二轮重点攻克
5. **每天固定时间**，形成习惯

**APP设置建议（墨墨背单词）**：

```
每日新词：100
复习上限：200
模式：认识/不认识（不用拼写）
```

#### 长难句训练

**每日练习**：
- 分析2-3个长难句
- 找出主干（主谓宾）
- 识别从句类型
- 翻译成中文

**长难句分析步骤**：
```
1. 找谓语动词（确定主干）
2. 找主语（谓语前的名词/代词）
3. 识别从句引导词（that/which/who/when/if等）
4. 确定从句类型（定语/状语/宾语从句）
5. 拆分并翻译
```

**常见句式结构**：
- 主语从句
- 宾语从句
- 定语从句（限制性/非限制性）
- 状语从句（时间/条件/原因/让步/目的）
- 同位语从句
- 强调句
- 倒装句
- 虚拟语气

**目标**：精做真题阅读，掌握题型规律

| 时间段 | 任务 | 时长 |
|--------|------|------|
| 早间 | 单词复习 | 30分钟 |

#### 阅读理解训练

**真题使用方法**（四步精读法）：

```
第一步：限时完成（15-18分钟/篇）
第二步：逐句翻译全文
第三步：分析每道题的答案来源
第四步：总结生词和长难句
```

**六大题型及策略**：

| 题型 | 特征 | 策略 |
|------|------|------|
| 主旨题 | What is the main idea/best title | 关注首尾段、各段首句 |
| 细节题 | According to the passage | 定位原文，同义替换 |
| 推理题 | It can be inferred/implied | 合理推理，不过度 |
| 词义题 | The word "X" refers to | 上下文语境判断 |
| 态度题 | Author's attitude | 找情感词汇 |
| 例证题 | Example is used to | 例子服务于论点 |

**干扰项特征**：
- 偷换概念
- 以偏概全
- 无中生有
- 过度推断
- 正话反说

#### 作文训练

**小作文（应用文）类型**：
- 书信类：建议信、投诉信、感谢信、道歉信、邀请信、推荐信
- 通知、备忘录

**大作文（图表/图画作文）结构**：
```
第一段：描述图表/图画（约50词）
第二段：分析原因/意义（约100词）
第三段：总结/建议（约50词）
```

## 薄弱基础提升策略

### 词汇突破（最重要）

**你的情况**：基础薄弱，词汇量可能不足2000

**建议策略**：
1. **降低日目标**：不求多，求记牢，每天20-25个新词
2. **高频词优先**：先背真题高频词2000
3. **多次重复**：一个词至少见7次以上
4. **利用碎片时间**：通勤、午休时用APP复习

**推荐APP**：
- 墨墨背单词（艾宾浩斯） - **主力推荐，只用这一个即可**
- 不背单词（语境记忆）
- 扇贝单词

#### 墨墨背单词使用指南

**初始设置**：
```
下载安装 → 注册账号 → 选择词书（搜索"考研"）
```

**核心设置（设置 → 学习设置）**：

| 设置项 | 推荐值 | 说明 |
|--------|--------|------|
| 每日新词上限 | 100 | 快速过词方案 |
| 每日复习上限 | 300 | 不限制复习 |
| 学习模式 | 看词选义 | 不用拼写，提高速度 |

**每日学习流程**：
```
打开APP → 先完成【复习】→ 再学习【新词】→ 完成当日任务
```

**推荐背单词时间（根据你的作息）**：

你的时间：工作日 10:00-19:30 上班，20:30-9:30 可用

| 时段 | 任务 | 时长 | 说明 |
|------|------|------|------|
| **早起 7:00-7:30** | 复习旧词 | 30分钟 | 早晨记忆效果最好 |
| 通勤路上 | 继续复习 | 碎片时间 | 如果有地铁/公交 |
| **晚间 21:00-22:00** | 学习新词 | 50-60分钟 | 下班后第一件事 |

**为什么推荐早晚分开**：
1. **早上复习**：睡眠后记忆巩固，早晨复习效果好
2. **晚上新学**：学完睡觉，睡眠帮助记忆固化
3. **分散学习**：比一次性学1.5小时效果更好

**操作技巧**：
- 每个单词2-3秒内做决定，不要纠结
- 认识 → 点"认识"；不认识 → 看释义后点"不认识"
- 先复习后新学，复习比新学更重要
- 每天必须打卡，断一天复习量会堆积

### 阅读突破

**你的情况**：可能读不懂文章，做题靠蒙

**建议策略**：
1. **先攻词汇**：词汇量不够，阅读无从谈起
2. **从简单开始**：先做2005年以前的真题
3. **精读为主**：每篇文章逐句翻译
4. **建立错题本**：记录每道错题的原因

### 写作突破

**你的情况**：可能不会写或写不出

**建议策略**：
1. **先背后写**：先背诵范文，培养语感
2. **模板为主**：英语薄弱者，模板是最快提分方式
3. **多写多改**：每周至少写1篇大作文
