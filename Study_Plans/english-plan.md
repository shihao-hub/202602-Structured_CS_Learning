# 英语一学习计划

> 英语一是考研中分值高且拉分明显的科目，超级薄弱需要持续投入

## 考试题型与分值分布

| 题型 | 题数 | 分值 | 难度 | 目标得分 |
|------|------|------|------|----------|
| 完形填空 | 20题 | 10分 | 较难 | 6-7分 |
| 阅读理解A | 20题 | 40分 | 难 | 28-32分 |
| 阅读理解B(新题型) | 5题 | 10分 | 中等 | 6-8分 |
| 翻译 | 5题 | 10分 | 较难 | 6-7分 |
| 小作文 | 1题 | 10分 | 中等 | 7-8分 |
| 大作文 | 1题 | 20分 | 中等 | 12-14分 |
| **总计** | - | **100分** | - | **65-76分** |

## 目标分析

当前水平：超级薄弱
目标分数：70分

**分数分配策略**：
- 阅读理解（50分）：目标34-40分（阅读是大头，必须重点突破）
- 写作（30分）：目标19-22分（可以通过模板提分）
- 完形+翻译（20分）：目标12-14分（难度大，保底即可）

## 学习阶段规划

### 基础阶段（2月下旬 - 5月底）

**目标**：词汇量突破5500，长难句能正确分析

#### 每日任务安排

| 时间段 | 任务 | 时长 |
|--------|------|------|
| 早间 | 背单词（新词+复习） | 45分钟 |
| 晚间 | 长难句分析 | 30分钟 |
| 晚间 | 阅读精读 | 45分钟 |
| **总计** | | **2小时** |

#### 词汇学习计划

**第1-4月：核心词汇**

| 周次 | 目标词汇量 | 日均新词 | 复习词数 |
|------|-----------|----------|----------|
| Week 1-4 | 800词 | 30词 | 60词 |
| Week 5-8 | 1600词 | 30词 | 90词 |
| Week 9-12 | 2400词 | 30词 | 120词 |
| Week 13-16 | 3200词 | 30词 | 150词 |

**艾宾浩斯复习周期**：
```
Day 1: 学习新词
Day 2: 第1次复习
Day 4: 第2次复习
Day 7: 第3次复习
Day 15: 第4次复习
Day 30: 第5次复习
```

**词汇学习方法**：
1. **词根词缀法**：掌握常见词根词缀
2. **语境记忆**：在句子中记单词
3. **多感官记忆**：看、读、写、听结合
4. **高频优先**：优先记忆真题高频词

#### 长难句训练

**每日练习**：
- 分析2-3个长难句
- 找出主干（主谓宾）
- 识别从句类型
- 翻译成中文

**长难句分析步骤**：
```
1. 找谓语动词（确定主干）
2. 找主语（谓语前的名词/代词）
3. 识别从句引导词（that/which/who/when/if等）
4. 确定从句类型（定语/状语/宾语从句）
5. 拆分并翻译
```

**常见句式结构**：
- 主语从句
- 宾语从句
- 定语从句（限制性/非限制性）
- 状语从句（时间/条件/原因/让步/目的）
- 同位语从句
- 强调句
- 倒装句
- 虚拟语气

### 强化阶段（6月 - 9月）

**目标**：精做真题阅读，掌握题型规律

#### 每日任务安排

| 时间段 | 任务 | 时长 |
|--------|------|------|
| 早间 | 单词复习 | 30分钟 |
| 晚间 | 真题阅读精做 | 90分钟 |
| 周末 | 作文练习 | 60分钟 |

#### 阅读理解训练

**真题使用方法**（四步精读法）：

```
第一步：限时完成（15-18分钟/篇）
第二步：逐句翻译全文
第三步：分析每道题的答案来源
第四步：总结生词和长难句
```

**六大题型及策略**：

| 题型 | 特征 | 策略 |
|------|------|------|
| 主旨题 | What is the main idea/best title | 关注首尾段、各段首句 |
| 细节题 | According to the passage | 定位原文，同义替换 |
| 推理题 | It can be inferred/implied | 合理推理，不过度 |
| 词义题 | The word "X" refers to | 上下文语境判断 |
| 态度题 | Author's attitude | 找情感词汇 |
| 例证题 | Example is used to | 例子服务于论点 |

**干扰项特征**：
- 偷换概念
- 以偏概全
- 无中生有
- 过度推断
- 正话反说

#### 作文训练

**小作文（应用文）类型**：
- 书信类：建议信、投诉信、感谢信、道歉信、邀请信、推荐信
- 通知、备忘录

**大作文（图表/图画作文）结构**：
```
第一段：描述图表/图画（约50词）
第二段：分析原因/意义（约100词）
第三段：总结/建议（约50词）
```

### 冲刺阶段（10月 - 12月）

**目标**：模拟训练，作文模板熟练，查漏补缺

#### 每日任务安排

| 时间段 | 任务 | 时长 |
|--------|------|------|
| 早间 | 单词巩固 | 20分钟 |
| 晚间 | 真题二刷/模拟 | 60分钟 |
| 晚间 | 作文背诵/练习 | 40分钟 |

#### 作文模板准备

**必备模板数量**：
- 小作文：各类型1个模板
- 大作文：3-5个万能模板

**模板练习方法**：
```
1. 整理并记忆模板
2. 用真题题目练习替换
3. 限时写作（小作文15分钟，大作文40分钟）
4. 找人批改或自我对照
```

#### 真题二刷

- 重点做错题
- 分析错误原因
- 总结出题规律
- 保持手感

## 薄弱基础提升策略

### 词汇突破（最重要）

**你的情况**：基础薄弱，词汇量可能不足2000

**建议策略**：
1. **降低日目标**：不求多，求记牢，每天20-25个新词
2. **高频词优先**：先背真题高频词2000
3. **多次重复**：一个词至少见7次以上
4. **利用碎片时间**：通勤、午休时用APP复习

**推荐APP**：
- 墨墨背单词（艾宾浩斯）
- 不背单词（语境记忆）
- 扇贝单词

### 阅读突破

**你的情况**：可能读不懂文章，做题靠蒙

**建议策略**：
1. **先攻词汇**：词汇量不够，阅读无从谈起
2. **从简单开始**：先做2005年以前的真题
3. **精读为主**：每篇文章逐句翻译
4. **建立错题本**：记录每道错题的原因

### 写作突破

**你的情况**：可能不会写或写不出

**建议策略**：
1. **先背后写**：先背诵范文，培养语感
2. **模板为主**：英语薄弱者，模板是最快提分方式
3. **多写多改**：每周至少写1篇大作文

**核心提醒**：

1. **词汇是基础**：每天必须背单词，雷打不动
2. **真题是王道**：反复做真题，不要做模拟题
3. **阅读是大头**：阅读提分空间最大
4. **作文靠模板**：基础薄弱者，模板是最快提分方式
5. **坚持最重要**：英语提分慢，但坚持就会有收获

**目标**：从超级薄弱提升到70分，关键在于：
- 词汇量从不足2000提升到5000+
- 阅读正确率从50%提升到70%+
- 作文从不会写到能写出合格文章
