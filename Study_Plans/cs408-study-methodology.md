# 408 专业课学习方法论

> 基于个人经验总结：王道书籍（未看完、大题未做）+ 真题卷 = 110分
> 目标：完善学习流程，提升至 120 分

## 核心问题诊断

**之前的问题**：看完视频和概念后不想做题，直接进入下一章，导致知识吸收不牢

```
之前：看视频 → 看概念 → [跳过题目] → 下一章
                           ↑ 问题所在：没有输出
```

**解决方案**：拆成"边看边做"，每节看完立刻做题

## 三阶段学习法

### 第一阶段：基础阶段（视频 + 选择题）

**目标**：快速熟悉知识，建立整体框架

```
每节流程：
  看一节视频（30-40分钟）
      ↓
  立刻做这节选择题（15-20分钟）
      ↓
  对答案，标记错题（5-10分钟）
      ↓
  下一节
```

**关键原则**：
1. 视频是主力学习工具，理解概念和思路
2. 看完一节**必须做选择题**，这是之前缺失的关键步骤
3. 书本概念当字典用：做题遇到不懂的翻书查
4. 大题暂时跳过，基础阶段不做大题
5. **错题必须标记**，为强化阶段做准备

**不做大题的原因**：
- 基础不够做大题会很挫败
- 选择题足以检验基础概念的掌握
- 大题留给有框架认知后再攻破，效率更高

### 第二阶段：强化阶段（看书 + 大题）

**目标**：深入理解，补充细节，攻克大题

**前提**：基础阶段全部完成，对所有知识有整体认识

```
每章流程：
  看书（细读概念，重点看基础阶段薄弱的点）
      ↓
  快速过这章的选择题错题（只刷错题，不全部重做）
      ↓
  做大题（先尝试写 → 对答案 → 理解思路）
      ↓
  下一章
```

**关键原则**：
1. 不再看视频，直接看书
2. 书要仔细看，补充视频没讲到的细节
3. 选择题不全部重做，**只刷第一轮标记的错题**
4. 大题：先自己写 → 对答案 → 不会的看思路 → 总结套路

**为什么这个顺序有效**：
- 第一轮已有框架，看书是在框架上补细节，效率极高
- 错题代表薄弱点，只刷错题是最高效的选择题复习方式
- 有了整体认知后做大题，不会像一开始那么迷茫

### 第三阶段：真题阶段（真题卷 + 书当字典）

**目标**：应试训练，查漏补缺

```
每套真题流程：
  限时完成（150分钟）
      ↓
  对答案，统计各科得分
      ↓
  逐题分析错因，翻书查对应知识点
      ↓
  整理错题知识点
```

**关键原则**：
1. 严格限时，模拟考试环境
2. 分析每道错题的错因（概念不清 / 计算错误 / 审题失误）
3. 书的角色是字典：哪里不会查哪里
4. 重点关注反复出错的知识点

## 三个阶段的关系

```
基础阶段         强化阶段         真题阶段
视频+选择题  →   看书+大题    →   真题卷
建立框架         深入理解         应试训练
  (求广度)        (求深度)        (求熟练)
```

## 工具定位

| 工具 | 基础阶段 | 强化阶段 | 真题阶段 |
|------|----------|----------|----------|
| 视频 | 主力学习 | 不再看 | 不再看 |
| 王道书概念 | 当字典查 | 仔细看 | 当字典查 |
| 选择题 | 每节必做 | 只刷错题 | 不做 |
| 大题 | 不做 | 每章必做 | 不做 |
| 真题卷 | 不做 | 不做 | 限时训练 |

## 铁律

1. **看完一节视频，选择题必须做**（基础阶段）
2. **错题必须标记**（贯穿始终）
3. **先选择后大题，先框架后细节**（学习顺序）
