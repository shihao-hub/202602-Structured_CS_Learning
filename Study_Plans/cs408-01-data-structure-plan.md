## 学习阶段规划

### 基础阶段（2月下旬 - 4月）

**目标**：掌握数据结构基本概念，能手写核心算法

#### 第1周：绑论与线性表

| 内容 | 学习要点 |
|------|----------|
| 基本概念 | 数据结构、算法、时间/空间复杂度 |
| 顺序表 | 静态/动态分配、插入/删除/查找 |
| 链表 | 单链表、双链表、循环链表 |
| 链表操作 | 建表、插入、删除、逆置、合并 |

**必会代码**：
- [ ] 顺序表插入、删除
- [ ] 单链表头插法、尾插法建表
- [ ] 单链表逆置
- [ ] 两个有序链表合并

**时间复杂度速查**：

| 操作 | 顺序表 | 链表 |
|------|--------|------|
| 按值查找 | O(n) | O(n) |
| 按位查找 | O(1) | O(n) |
| 插入 | O(n) | O(1)* |
| 删除 | O(n) | O(1)* |

*链表插入删除O(1)是指已知位置的情况

#### 第2周：栈、队列和数组

| 内容 | 学习要点 |
|------|----------|
| 栈 | 顺序栈、链栈、共享栈 |
| 栈应用 | 括号匹配、表达式求值、递归 |
| 队列 | 顺序队列、循环队列、链队列 |
| 双端队列 | 输入受限、输出受限 |
| 特殊矩阵 | 对称矩阵、三角矩阵、三对角矩阵、稀疏矩阵 |

**必会代码**：
- [ ] 栈的基本操作（顺序栈实现）
- [ ] 循环队列判空、判满、入队、出队
- [ ] 括号匹配算法
- [ ] 表达式转换（中缀→后缀）

**循环队列公式**：
```
队空：front == rear
队满：(rear + 1) % MaxSize == front
队长：(rear - front + MaxSize) % MaxSize
入队：rear = (rear + 1) % MaxSize
出队：front = (front + 1) % MaxSize
```

#### 第3周：串

| 内容 | 学习要点 |
|------|----------|
| 串存储 | 定长顺序、堆分配、块链 |
| 模式匹配 | 朴素算法、KMP算法 |
| next数组 | 手算方法、改进的nextval |

**必会代码**：
- [ ] 朴素模式匹配
- [ ] KMP算法
- [ ] 求next数组
- [ ] 求nextval数组

**KMP next数组手算**：
```
模式串：a b a a b c a c
next值：0 1 1 2 2 3 1 2
（注：next[1]=0 或 next[0]=-1，看题目约定）
```

#### 第4-5周：树与二叉树

| 内容 | 学习要点 |
|------|----------|
| 树基础 | 度、高度、路径、性质 |
| 二叉树 | 满二叉树、完全二叉树、性质 |
| 二叉树存储 | 顺序存储、链式存储 |
| 遍历 | 先序、中序、后序、层序 |
| 线索二叉树 | 概念、构造、遍历 |
| 树的存储 | 双亲表示、孩子表示、孩子兄弟表示 |
| 树/森林遍历 | 与二叉树遍历的对应关系 |
| 哈夫曼树 | 构造、WPL、哈夫曼编码 |

**必会代码**：
- [ ] 先序/中序/后序遍历（递归+非递归）
- [ ] 层序遍历
- [ ] 求树高
- [ ] 由遍历序列构造二叉树
- [ ] 线索二叉树的构造

**二叉树性质**：
```
1. 第i层最多 2^(i-1) 个结点
2. 高度为h的树最多 2^h - 1 个结点
3. n₀ = n₂ + 1（叶结点 = 度为2的结点 + 1）
4. 完全二叉树：n个结点，高度 = ⌊log₂n⌋ + 1
```

#### 第6周：图

| 内容 | 学习要点 |
|------|----------|
| 图基础 | 有向图、无向图、度、路径、连通性 |
| 图存储 | 邻接矩阵、邻接表、十字链表、邻接多重表 |
| 图遍历 | DFS、BFS、遍历序列 |
| 最小生成树 | Prim算法、Kruskal算法 |
| 最短路径 | Dijkstra算法、Floyd算法 |
| 拓扑排序 | 算法、判断DAG |
| 关键路径 | 事件最早/最晚时间、关键活动 |

**必会代码**：
- [ ] DFS、BFS（邻接表/邻接矩阵）
- [ ] Prim算法
- [ ] Kruskal算法
- [ ] Dijkstra算法
- [ ] 拓扑排序

**算法复杂度**：

| 算法 | 时间复杂度 |
|------|------------|
| DFS/BFS（邻接表） | O(V+E) |
| DFS/BFS（邻接矩阵） | O(V²) |
| Prim（邻接矩阵） | O(V²) |
| Kruskal | O(ElogE) |
| Dijkstra | O(V²) 或 O(ElogV) |
| Floyd | O(V³) |
| 拓扑排序 | O(V+E) |

#### 第7周：查找

| 内容 | 学习要点 |
|------|----------|
| 顺序查找 | 一般、有序表 |
| 折半查找 | 算法、判定树、ASL |
| 分块查找 | 索引顺序查找 |
| BST | 插入、删除、查找、ASL |
| 平衡二叉树 | LL/RR/LR/RL调整 |
| 红黑树 | 性质（了解） |
| B树/B+树 | 性质、插入、删除 |
| 散列表 | 构造、冲突处理、ASL |

**必会代码**：
- [ ] 折半查找
- [ ] BST的查找、插入
- [ ] AVL树的旋转调整

**ASL计算**：
```
查找成功 ASL = Σ(第i层结点数 × i) / 总结点数
查找失败 ASL = Σ(到达第i层失败结点的比较次数 × 失败结点数) / 失败结点总数
```

#### 第8周：排序

| 内容 | 学习要点 |
|------|----------|
| 插入排序 | 直接插入、折半插入、希尔排序 |
| 交换排序 | 冒泡排序、快速排序 |
| 选择排序 | 简单选择、堆排序 |
| 归并排序 | 二路归并 |
| 基数排序 | LSD、MSD |
| 外部排序 | 多路归并、败者树、置换选择 |

**必会代码**：
- [ ] 直接插入排序
- [ ] 快速排序（划分过程）
- [ ] 堆调整（建堆、堆排序）
- [ ] 归并排序

**排序算法对比**：

| 算法 | 平均 | 最好 | 最坏 | 空间 | 稳定 |
|------|------|------|------|------|------|
| 直接插入 | O(n²) | O(n) | O(n²) | O(1) | Y |
| 冒泡 | O(n²) | O(n) | O(n²) | O(1) | Y |
| 快速排序 | O(nlogn) | O(nlogn) | O(n²) | O(logn) | N |
| 简单选择 | O(n²) | O(n²) | O(n²) | O(1) | N |
| 堆排序 | O(nlogn) | O(nlogn) | O(nlogn) | O(1) | N |
| 归并 | O(nlogn) | O(nlogn) | O(nlogn) | O(n) | Y |
| 基数 | O(d(n+r)) | O(d(n+r)) | O(d(n+r)) | O(r) | Y |

## 学习方法建议

### 数据结构学习要点

1. **必须手写代码**：每个核心算法至少手写3遍
2. **理解原理**：不要死记代码，理解思想
3. **复杂度分析**：每个算法都要能分析复杂度
4. **画图辅助**：数据结构的变化用图来理解

### 代码练习建议

```
每日练习：
1. 默写1-2个核心算法
2. 分析时间/空间复杂度
3. 思考变形题目
```
