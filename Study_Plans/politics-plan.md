# 政治学习计划

> 政治是考研中性价比较高的科目，后期发力即可

## 考试题型与分值分布

| 题型 | 题数 | 分值 | 说明 |
|------|------|------|------|
| 单选题 | 16题 | 16分 | 每题1分 |
| 多选题 | 17题 | 34分 | 每题2分，漏选得1分 |
| 分析题 | 5题 | 50分 | 每题10分 |
| **总计** | 38题 | **100分** | |

## 考试内容与分值

| 科目 | 简称 | 分值占比 | 特点 |
|------|------|----------|------|
| 马克思主义基本原理 | 马原 | 约24分 | 理论性强，需要理解 |
| 毛泽东思想和中国特色社会主义理论 | 毛中特 | 约30分 | 内容最多，时政结合 |
| 中国近现代史纲要 | 史纲 | 约14分 | 时间线为主 |
| 思想道德与法治 | 思法 | 约16分 | 相对简单 |
| 形势与政策、当代世界经济与政治 | 时政 | 约16分 | 需要关注时事 |

## 目标分析

当前水平：正常准备能65分
目标分数：65-70分

**分数分配策略**：
- 选择题（50分）：目标35-40分（选择题决定分数线）
- 分析题（50分）：目标30-35分（大家分数相近）

**关键认知**：
- 政治拉分主要在选择题
- 分析题大家分数差距不大（25-35分区间）
- 多选题是难点和拉分点

## 学习阶段规划

### 前期准备（2月 - 7月）

**建议**：这个阶段以专业课和英语为主，政治可以：
- 偶尔看看新闻了解时政
- 有空可以翻翻马原（理论性强，需要理解）
- 不需要系统学习

### 基础阶段（8月 - 9月）

**目标**：过一遍知识点，配合1000题

#### 学习安排

| 科目 | 建议时长 | 重点 |
|------|----------|------|
| 马原 | 2-3周 | 唯物论、辩证法、认识论、唯物史观 |
| 毛中特 | 3-4周 | 各时期理论、新时代内容 |
| 史纲 | 1-2周 | 时间线、重要会议、历史意义 |
| 思法 | 1周 | 相对简单，过一遍即可 |

#### 每日任务

```
每日政治学习：2-3小时

第一轮（听课+做题）：
1. 看视频课（1-1.5小时）
2. 做对应1000题（0.5-1小时）
3. 整理错题笔记（0.5小时）
```

#### 马原重点概念

**唯物论**：
- 物质与意识的关系
- 物质的唯一特性：客观实在性
- 意识的能动作用

**辩证法**（三大规律）：
- 对立统一规律（核心）
- 质量互变规律
- 否定之否定规律

**认识论**：
- 实践与认识的关系
- 真理的绝对性与相对性
- 认识的反复性和无限性

**唯物史观**：
- 社会存在与社会意识
- 生产力与生产关系
- 经济基础与上层建筑
- 人民群众是历史的创造者

### 强化阶段（10月）

**目标**：二刷错题，强化薄弱点

#### 学习安排

```
每日政治学习：2-3小时

1. 二刷1000题错题（1小时）
2. 整理重点知识点（1小时）
3. 做真题选择题（1小时）
```

#### 多选题突破

**多选题陷阱**：
- 说法太绝对（"完全"、"只有"、"必然"）
- 概念偷换
- 以偏概全
- 因果颠倒

**多选题技巧**：
- 先排除明显错误选项
- 注意题干限定词
- 相互矛盾的选项通常不会同时正确
- 不确定时，宁可少选

### 冲刺阶段（11月 - 12月）

**目标**：背诵分析题，模拟训练

#### 11月任务

```
每日政治学习：3-4小时

1. 肖八选择题（1小时）
2. 分析题答题框架整理（1小时）
3. 时政热点整理（0.5小时）
4. 查漏补缺（0.5-1小时）
```

#### 12月任务

```
每日政治学习：4-5小时

1. 肖四选择题+分析题（2小时）
2. 背诵肖四分析题（2小时）
3. 模拟训练（1小时）
```

#### 分析题答题技巧

**答题结构**：
```
1. 原理（是什么）：点明相关原理/理论
2. 分析（为什么）：结合材料分析
3. 总结（怎么做）：联系实际或提出建议
```

**答题要求**：
- 字迹工整
- 分点作答
- 结合材料
- 观点明确

**分析题背诵优先级**：
1. 肖四（必背）
2. 肖八分析题（选背）
3. 其他押题资料（参考）

## 各科目学习要点

### 马原（约24分）

**特点**：理论性强，需要理解

**学习建议**：
- 理解原理，而非死记
- 掌握哲学术语
- 多做选择题练手感

**高频考点**：
- 矛盾的普遍性与特殊性
- 实践与认识的辩证关系
- 真理与价值
- 社会基本矛盾运动

### 毛中特（约30分）

**特点**：内容最多，与时政结合紧密

**学习建议**：
- 把握各时期理论特点
- 重点关注新时代内容
- 结合时政理解

**高频考点**：
- 新民主主义革命理论
- 社会主义改造理论
- 中国特色社会主义理论体系
- 习近平新时代中国特色社会主义思想

### 史纲（约14分）

**特点**：时间线清晰，记忆为主

**学习建议**：
- 建立时间线
- 记住重要会议和意义
- 理解历史发展规律

**重要时间节点**：
```
1840 鸦片战争
1919 五四运动
1921 中共成立
1949 新中国成立
1956 社会主义改造完成
1978 改革开放
```

### 思法（约16分）

**特点**：相对简单，常识性强

**学习建议**：
- 过一遍即可
- 注意法律相关知识
- 结合社会热点

### 时政（约16分）

**特点**：需要关注时事

**学习建议**：
- 11月后集中看时政资料
- 关注重大会议、领导人讲话
- 国际热点事件

**核心提醒**：

1. **选择题是关键**：选择题决定分数线，分析题大家差不多
2. **多选题是难点**：需要大量练习
3. **肖四必背**：分析题押题准确率高
4. **不要太早开始**：8月开始完全来得及
5. **保持时政敏感**：关注国内外大事
