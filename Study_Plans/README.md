## 核心策略

### 数学一（超级薄弱 -> 120+）

1. **不跳步**：每一章都要扎实，基础不牢会影响后续
2. **多做题**：数学是做出来的，不是看出来的
3. **建错题本**：记录典型错题和思路
4. **定期复习**：每周回顾本周错题

### 英语一（超级薄弱 -> 70+）

1. **词汇为王**：每天雷打不动背单词（艾宾浩斯复习）
2. **长难句突破**：每天分析2-3个长难句
3. **阅读真题**：真题为王，反复精读
4. **作文模板**：整理个人模板，考前熟练背诵

### 408（110 -> 120+）

1. **查漏补缺**：找出之前薄弱的知识点
2. **代码实现**：核心算法必须能手写
3. **真题训练**：408真题质量极高，反复做
4. **跨科联系**：注意四门课之间的关联

### 政治（正常 -> 65+）

1. **后期发力**：8月后开始系统学习
2. **选择题为王**：选择题决定分数线
3. **时政关注**：平时适当关注新闻
4. **肖四肖八**：考前必背
