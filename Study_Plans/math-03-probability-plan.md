**重点题型**：
- [ ] 古典概型计算

#### 第2周：一维随机变量

| 内容 | 学习要点 |
|------|----------|
| 分布函数 | 定义、性质 |
| 离散型分布 | 0-1分布、二项分布、泊松分布、几何分布 |
| 连续型分布 | 均匀分布、指数分布、正态分布 |
| 函数分布 | Y=g(X)的分布 |

**重点题型**：
- [ ] 求分布函数
- [ ] 常见分布识别
- [ ] 一维随机变量函数的分布
- [ ] 分布函数的应用

**重要分布公式**：

| 分布 | 记号 | 期望 | 方差 |
|------|------|------|------|
| 0-1分布 | B(1,p) | p | p(1-p) |
| 二项分布 | B(n,p) | np | np(1-p) |
| 泊松分布 | P(λ) | λ | λ |
| 均匀分布 | U(a,b) | (a+b)/2 | (b-a)²/12 |
| 指数分布 | E(λ) | 1/λ | 1/λ² |
| 正态分布 | N(μ,σ²) | μ | σ² |

#### 第5周：数字特征

| 内容 | 学习要点 |
|------|----------|
| 期望 | 定义、性质、函数期望 |
| 方差 | 定义、性质、计算公式 |
| 协方差与相关系数 | 定义、性质、独立与不相关 |
| 矩 | 原点矩、中心矩 |

**重点题型**：
- [ ] 期望、方差计算
- [ ] 协方差、相关系数计算
- [ ] 独立与不相关的关系
- [ ] 数字特征综合应用

**重要性质**：

```
E(aX+bY) = aE(X) + bE(Y)
D(X) = E(X²) - [E(X)]²
D(aX+b) = a²D(X)
D(X±Y) = D(X) + D(Y) ± 2Cov(X,Y)
独立 → 不相关，反之不一定成立
```

#### 第6周：大数定律与中心极限定理

**重点题型**：

#### 第7周：数理统计

| 内容 | 学习要点 |
|------|----------|
| 基本概念 | 总体、样本、统计量 |
| 三大分布 | χ²分布、t分布、F分布 |
| 抽样分布 | 样本均值、样本方差的分布 |

**重要抽样分布定理**：

设 X₁,...,Xₙ 来自 N(μ,σ²)：

| 统计量 | 分布 |
|--------|------|
| X̄ | N(μ, σ²/n) |
| (n-1)S²/σ² | χ²(n-1) |
| (X̄-μ)/(S/√n) | t(n-1) |

## 学习方法建议

### 概率学习特点

1. **计算为主**：公式必须熟练
2. **模型识别**：快速识别分布类型
3. **积分运算**：与高数联系紧密
4. **步骤规范**：大题要求过程完整

### 高效学习策略

1. **熟记分布表**：8种常见分布的期望、方差
2. **掌握计算套路**：期望、方差、协方差的计算
3. **多做二维题**：这是必考且难点
4. **注意与高数结合**：积分、级数常用

### 易错点提醒

- 分布函数是"≤"，不是"<"
- P(X=a) 对连续型随机变量为0
- 独立≠不相关（仅正态分布时等价）
- 样本方差除以 n-1，不是 n
- MLE求解时注意定义域
