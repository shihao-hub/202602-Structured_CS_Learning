# 考研英语一学习模块

## 模块概述

本模块为考研英语一备考提供系统化学习资源，重点聚焦**词汇学习方法论**和**阅读理解策略**。

### 核心理念

- **方法论优先**：不提供简单的词汇列表，而是教授科学的记忆方法和学习框架
- **策略驱动**：针对考研英语一的阅读理解题型，提供系统化解题策略
- **实战导向**：所有内容围绕考研英语一真题命题规律设计

## 快速开始

### 1. 交互式学习模式

直接运行主程序，通过菜单选择学习内容：

```bash
python main.py
```

**菜单选项：**
- 词汇学习模式（随机测试、艾宾浩斯复习提醒、错题本）
- 阅读策略查看（显示阅读理解方法论概览）
- 学习统计（词汇学习进度、正确率）
- 退出

### 2. 命令行快捷模式

```bash
# 直接进入词汇学习
python main.py --vocab

# 查看阅读策略
python main.py --reading

# 帮助信息
python main.py --help
```

## 词汇学习子模块

### 核心内容

1. **科学记忆方法**
   - 艾宾浩斯遗忘曲线复习计划
   - 词根词缀记忆法（30+核心词根）
   - 语境记忆法（搭配+例句）

2. **分级学习路线**
   - 基础词汇（高中3500词）巩固
   - 四六级衔接词汇
   - 考研新增词汇（约2000词）

3. **主题化高频词汇**
   - 科技类、经济类、社会类、教育类等8大主题
   - 每个主题15-20个核心词汇
   - 常见搭配和用法

### 工具功能（vocabulary_tool.py）

- 随机词汇测试（中→英 / 英→中）
- 艾宾浩斯复习提醒（基于学习日期自动提醒）
- 错题本管理（自动保存到 `user_data/` 目录）
- 学习进度统计
- 词汇搜索功能
- 数据导入/导出

**使用示例：**

```python
# 可独立运行
python vocabulary/vocabulary_tool.py

# 或在代码中导入
from vocabulary.vocabulary_tool import VocabularyTool
tool = VocabularyTool()
tool.start_quiz()
```

## 阅读理解子模块

### 核心内容

1. **阅读三步法**
   - Skimming（略读）：把握主旨
   - Scanning（扫读）：定位关键信息
   - Close Reading（精读）：理解细节逻辑

2. **长难句分析**
   - 主谓宾识别
   - 从句类型判断
   - 修饰成分剥离技巧

3. **六大题型解题策略**
   - 主旨大意题
   - 细节题
   - 推理判断题
   - 词义猜测题
   - 态度观点题
   - 结构题（例证题）

4. **文章结构识别**
   - 5种常见结构模式（问题-解决型、现象-解释型等）
   - 信号词识别
   - 针对性阅读策略

5. **真题练习方法论**
   - 精做真题四步法
   - 时间管理策略（每篇15-18分钟）
   - 干扰项特征识别

### 推荐学习路径

```
1. 阅读 reading_strategies_guide.md （建立整体认知）
   ↓
2. 学习 question_types_analysis.md （掌握各题型解法）
   ↓
3. 阅读 article_structure_guide.md （提升文章把控能力）
```

## 学习建议

### 词汇学习

- **每日新词量**：20-30个（结合个人基础调整）
- **复习优先**：严格按照艾宾浩斯曲线复习（比学新词更重要！）
- **多维记忆**：词根词缀+语境+搭配，三管齐下
- **避免陷阱**：不要试图"从A背到Z"，按主题和词根分组记忆

### 阅读训练

- **前期（基础阶段）**：每天1篇精读，重点分析长难句
- **中期（强化阶段）**：每天2篇限时训练，总结题型规律
- **后期（冲刺阶段）**：每天4篇模拟（完整套题），时间控制在60-70分钟
- **真题为王**：近10年真题至少做3遍（第1遍限时，第2遍逐句翻译，第3遍总结错因）

### 时间规划

| 阶段 | 时间 | 词汇任务 | 阅读任务 |
|------|------|----------|----------|
| 基础阶段 | 3-6月 | 完成5500词第一轮 | 每天1篇精读 |
| 强化阶段 | 7-9月 | 熟词僻义+高频词汇 | 每天2篇限时+真题 |
| 冲刺阶段 | 10-12月 | 高频词汇反复复习 | 每天4篇模拟套题 |
