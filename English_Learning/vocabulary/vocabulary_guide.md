## 1. 科学记忆原理：艾宾浩斯遗忘曲线

### 1.1 遗忘规律

德国心理学家艾宾浩斯（Hermann Ebbinghaus）研究发现，人类大脑对新事物的遗忘呈现以下规律：

```
记忆保持率随时间变化：

20分钟后 → 保留 58.2%  (遗忘 41.8%)
1小时后  → 保留 44.2%  (遗忘 55.8%)
1天后    → 保留 33.7%  (遗忘 66.3%) ⚠️ 关键点！
2天后    → 保留 27.8%  (遗忘 72.2%)
6天后    → 保留 25.4%  (遗忘 74.6%)
31天后   → 保留 21.1%  (遗忘 78.9%)
```

**核心结论：**
- **遗忘速度先快后慢**：第1天遗忘最多（66.3%）
- **及时复习至关重要**：越早复习，巩固效果越好

---

### 1.2 科学复习周期

根据遗忘曲线，最佳复习时间点为：

| 复习轮次 | 间隔时间 | 目的 |
|---------|---------|------|
| 第1次 | 学习后**当天晚上** | 巩固短期记忆 |
| 第2次 | 学习后**第2天** | 对抗第一波遗忘高峰 |
| 第3次 | 学习后**第4天** | 强化中期记忆 |
| 第4次 | 学习后**第7天** | 转化为长期记忆 |
| 第5次 | 学习后**第15天** | 确保持久记忆 |
| 第6次 | 学习后**第30天** | 终极巩固（可选） |

**实践建议：**

```
周一学习新词 20 个
  ├─ 周一晚上：第1次复习（20个）
  ├─ 周三：第2次复习（20个）
  ├─ 周五：第3次复习（20个）
  ├─ 下周一：第4次复习（20个）
  └─ 两周后周一：第5次复习（20个）

同时：
  周二学习新词 20 个（按同样周期复习）
  周三学习新词 20 个（按同样周期复习）
  ...以此类推
```

**关键：** 每天既要学新词，又要复习多批旧词！

---

### 1.3 复习方法

#### 方法1：看英文回忆中文
- 翻开单词本，遮住中文释义
- 看到英文单词，立即在心中回忆中文意思
- 1-2秒内能想起 → PASS
- 想不起或犹豫 → 标记为需强化

#### 方法2：看中文回忆英文（更难，更有效）
- 遮住英文单词，只看中文释义
- 尝试拼写出英文单词
- 拼写正确 → PASS
- 拼写错误或遗忘 → 重点标记

#### 方法3：使用 vocabulary_tool.py
- 自动随机抽取复习单词
- 根据学习日期自动提醒复习
- 错题本功能，针对性强化

---

### 1.4 遗忘的本质与对策

**遗忘不是失败，而是正常现象！**

- **遗忘是大脑筛选机制**：大脑会遗忘不重要的信息
- **重复是告诉大脑"这很重要"**：通过复习强化神经连接
- **理解记忆优于机械记忆**：理解词根词缀，建立逻辑联系

**心理建设：**
- 今天背了明天忘 → 正常！第3次复习就会好很多
- 不要因为遗忘而焦虑 → 遵循科学规律，持续复习
- 质量>数量 → 宁可少背几个词，也要保证复习质量

## 2. 词根词缀记忆法（核心方法）

### 2.1 为什么词根词缀法有效？

英语单词 70-80% 由**词根 + 词缀**构成，类似中文的"偏旁部首"。

**举例：**

中文："江、河、湖、海、洋、泊、泳、洗" 都有"氵（水）"旁 → 都与水相关

英文：`inject, reject, project, eject` 都有 `-ject（投掷）` → 都与"投、掷"相关

**优势：**
- **举一反三**：掌握1个词根，可推测10个单词
- **提升阅读速度**：遇到生词可以猜测大致意思
- **长期记忆更牢固**：理解型记忆优于死记硬背

---

### 2.2 常见前缀（Prefixes）

前缀通常改变单词的**意义方向**或**程度**。

| 前缀 | 含义 | 示例 |
|------|------|------|
| **un-** | 不、非、相反 | unhappy(不快乐), unfair(不公平), undo(撤销) |
| **re-** | 再、重新、向后 | rewrite(重写), return(返回), review(复习) |
| **pre-** | 预先、在...之前 | preview(预览), predict(预测), prefix(前缀) |
| **dis-** | 不、相反、分离 | disagree(不同意), disappear(消失), dishonest(不诚实) |
| **mis-** | 错误、坏 | mistake(错误), misunderstand(误解), mislead(误导) |
| **in-/im-/il-/ir-** | 不、非 | inactive(不活跃), impossible(不可能), illegal(非法), irregular(不规则) |
| **anti-** | 反、抗 | antiwar(反战), antibiotic(抗生素), antisocial(反社会) |
| **de-** | 向下、去除、相反 | decrease(减少), devalue(贬值), decode(解码) |
| **ex-/e-** | 向外、出 | export(出口), exit(出口), emit(发出), eject(弹出) |
| **inter-** | 之间、相互 | international(国际的), interact(互动), interview(面试) |
| **sub-** | 在下、次 | subway(地铁), subtitle(字幕), submarine(潜水艇) |
| **trans-** | 横跨、转移 | transport(运输), translate(翻译), transform(转变) |
| **over-** | 过度、在上 | overwork(过度工作), overlook(俯瞰/忽视), overcome(克服) |
| **under-** | 在下、不足 | underground(地下), underestimate(低估), undergraduate(本科生) |
| **super-** | 超级、在上 | supermarket(超市), superhero(超级英雄), superior(优越的) |

**练习：**
- `dis` + `agree` = disagree（不同意）
- `re` + `turn` = return（返回）
- `pre` + `view` = preview（预览）

**真题高频前缀：** `un-`, `re-`, `dis-`, `in-`, `pre-`

### 2.3 常见后缀（Suffixes）

后缀通常决定单词的**词性**（名词、动词、形容词、副词）。

#### 2.3.1 名词后缀

| 后缀 | 含义 | 示例 |
|------|------|------|
| **-tion/-sion** | 动作、状态 | action(行动), decision(决定), education(教育) |
| **-ment** | 动作、结果 | development(发展), government(政府), achievement(成就) |
| **-ness** | 状态、性质 | happiness(快乐), kindness(善良), darkness(黑暗) |
| **-er/-or** | 做...的人/物 | teacher(教师), actor(演员), computer(计算机) |
| **-ist** | ...主义者/专家 | scientist(科学家), artist(艺术家), socialist(社会主义者) |
| **-ism** | 主义、学说 | socialism(社会主义), tourism(旅游业), criticism(批评) |
| **-ity/-ty** | 性质、状态 | ability(能力), society(社会), safety(安全) |
| **-ance/-ence** | 状态、性质 | importance(重要性), difference(差异), performance(表现) |

#### 2.3.2 形容词后缀

| 后缀 | 含义 | 示例 |
|------|------|------|
| **-ful** | 充满...的 | beautiful(美丽的), useful(有用的), hopeful(有希望的) |
| **-less** | 无...的、缺乏 | hopeless(无望的), careless(粗心的), homeless(无家可归的) |
| **-able/-ible** | 能够...的 | readable(可读的), possible(可能的), comfortable(舒适的) |
| **-ous/-ious** | 充满...的 | famous(著名的), dangerous(危险的), curious(好奇的) |
| **-ive** | ...的、有...性质 | active(活跃的), creative(创造性的), expensive(昂贵的) |
| **-al** | ...的、关于 | national(国家的), natural(自然的), cultural(文化的) |
| **-ic** | ...的、关于 | economic(经济的), scientific(科学的), historic(历史的) |

#### 2.3.3 动词后缀

| 后缀 | 含义 | 示例 |
|------|------|------|
| **-ize/-ise** | 使...化 | realize(实现), modernize(现代化), organize(组织) |
| **-fy/-ify** | 使成为 | simplify(简化), beautify(美化), classify(分类) |
| **-en** | 使成为 | widen(加宽), strengthen(加强), shorten(缩短) |

#### 2.3.4 副词后缀

| 后缀 | 含义 | 示例 |
|------|------|------|
| **-ly** | ...地（方式） | quickly(快速地), happily(快乐地), carefully(仔细地) |
| **-ward(s)** | 向...方向 | forward(向前), backward(向后), upward(向上) |

### 2.4 核心词根（Roots）

#### 高频词根列表

| 词根 | 含义 | 衍生词示例 |
|------|------|-----------|
| **-ject-** | 投掷 | inject(注入), reject(拒绝), project(投射), eject(弹出), subject(主题) |
| **-mit-/-miss-** | 发送、放出 | submit(提交), admit(承认), dismiss(解散), mission(任务), emit(发出) |
| **-cede-/-ceed-/-cess-** | 走、让步 | proceed(前进), exceed(超过), success(成功), access(进入), recede(后退) |
| **-duc-/-duct-** | 引导、带领 | introduce(介绍), produce(生产), conduct(引导), educate(教育), reduce(减少) |
| **-port-** | 携带、运输 | transport(运输), import(进口), export(出口), support(支持), portable(便携的) |
| **-pos-/-posit-** | 放置 | position(位置), compose(组成), dispose(处理), deposit(存款), oppose(反对) |
| **-spect-** | 看 | respect(尊重), inspect(检查), expect(期待), prospect(前景), suspect(怀疑) |
| **-scrib-/-script-** | 写 | describe(描述), prescribe(开处方), subscribe(订阅), manuscript(手稿) |
| **-vid-/-vis-** | 看见 | video(视频), visible(可见的), vision(视力), supervise(监督), television(电视) |
| **-aud-** | 听 | audience(观众), audio(音频), auditorium(礼堂), audible(可听见的) |
| **-dict-** | 说 | dictionary(字典), predict(预测), contradict(矛盾), dictate(口述) |
| **-fac-/-fact-/-fect-** | 做、制造 | factory(工厂), manufacture(制造), effect(效果), perfect(完美的), affect(影响) |
| **-form-** | 形状、形成 | reform(改革), inform(通知), transform(转变), uniform(制服), perform(表演) |
| **-press-** | 压 | pressure(压力), impress(印象), express(表达), depress(沮丧), compress(压缩) |
| **-tract-** | 拉、抽 | attract(吸引), contract(合同), extract(提取), distract(分散), subtract(减去) |
| **-struct-** | 建造 | structure(结构), construct(建造), instruct(指导), destroy(破坏), obstruct(阻碍) |

**详细示例：词根 `-ject-`（投掷）**

```
-ject- = throw (投掷)

1. inject (v.) = in-(向内) + ject(投掷) → 注入
   例句：inject medicine into the vein（向静脉注射药物）

2. reject (v.) = re-(向后) + ject(投掷) → 拒绝（扔回去）
   例句：reject the proposal（拒绝提议）

3. project (v./n.) = pro-(向前) + ject(投掷) → 投射；项目
   例句：project an image on the screen（在屏幕上投影图像）

4. eject (v.) = e-(向外) + ject(投掷) → 弹出、驱逐
   例句：eject the CD from the player（从播放器中弹出光盘）

5. subject (n./v.) = sub-(在下) + ject(投掷) → 主题；使服从
   例句：the subject of the essay（文章的主题）

6. object (n./v.) = ob-(相对) + ject(投掷) → 物体；反对
   例句：object to the plan（反对计划）
```

**学习方法：**
1. 每天学习 3-5 个词根
2. 每个词根掌握 5-10 个衍生词
3. 理解"前缀+词根+后缀"的组合逻辑
4. 在阅读中遇到新词时，尝试拆解词根

### 2.5 实战练习：拆解陌生单词

**练习1：** `unpredictable`

```
拆解：un- + pre- + dict + -able
      不   预先   说   能...的
含义：不能预先说的 → 不可预测的
```

**练习2：** `transportation`

```
拆解：trans- + port + -ation
      横跨    运输    名词后缀
含义：横跨运输 → 交通运输
```

**练习3：** `international`

```
拆解：inter- + nation + -al
      之间    国家    形容词后缀
含义：国家之间的 → 国际的
```

**技巧：** 即使不认识整个单词,通过词根词缀也能猜出大致意思！

## 3. 语境记忆法

### 3.1 为什么需要语境？

**问题：** 孤立背单词（word list）效率低，容易遗忘

**解决：** 在真实语境中学习单词

- **阅读文章**：通过真题阅读记忆单词
- **搭配记忆**：记住词汇的常见搭配（collocation）
- **例句记忆**：用经典例句强化记忆

---

### 3.2 搭配记忆法（Collocation）

**核心：** 记住单词的"好朋友"（常一起出现的词）

**示例：**

| 单词 | 常见搭配 |
|------|---------|
| **make** | make a decision(做决定), make progress(取得进步), make sense(有意义) |
| **take** | take action(采取行动), take responsibility(承担责任), take advantage of(利用) |
| **achieve** | achieve success(取得成功), achieve goals(实现目标), achieve balance(达到平衡) |
| **conduct** | conduct research(进行研究), conduct experiments(做实验), conduct business(做生意) |
| **raise** | raise awareness(提高意识), raise questions(提出问题), raise funds(筹集资金) |
| **address** | address the issue(解决问题), address the audience(向听众讲话) |
| **enhance** | enhance efficiency(提高效率), enhance quality(提升质量) |

**练习：** 用"achieve"造句
- ✅ achieve success（搭配正确）
- ❌ achieve victory（较少用，通常用 win victory）

---

### 3.3 例句记忆法

**原则：** 一个单词配一个经典例句（最好来自真题）

**示例：**

**单词：** `decline` (v.) 下降；拒绝

**例句：**
1. The number of students declined sharply last year.
   （去年学生数量急剧下降。）

2. He politely declined the invitation.
   （他礼貌地拒绝了邀请。）

**技巧：**
- 例句要短（10-15词）
- 例句要典型（体现单词核心用法）
- 朗读例句3-5遍，加深印象

---

### 3.4 真题阅读中学习词汇

**步骤：**

1. **第一遍：** 限时完成阅读题（不查词）
2. **第二遍：** 标出所有生词
3. **第三遍：** 查词典，理解生词在文章中的含义
4. **第四遍：** 摘抄生词+例句到笔记本
5. **第五遍：** 朗读全文，巩固生词

**优势：**
- 真题词汇最贴近考试
- 在语境中理解，记忆更深刻
- 同时提升阅读能力

## 4. 分级学习路线

### 4.1 词汇分级体系

考研英语词汇并非全是新词，合理规划学习路线很重要：

```
考研英语词汇体系：

第一层：高中词汇（约3500词）
         ↓ 基础必须牢固！
第二层：大学四六级词汇（约5000-6000词）
         ↓ 重点是考研未出现的
第三层：考研新增词汇（约2000词）
         ↓ 核心攻克对象
第四层：考研真题高频词（约500-800词）
         ↓ 反复强化
```

---

### 4.2 基础词汇巩固（高中3500词）

**自测方法：**

随机抽取100个高中词汇，测试能否：
- 看到英文立即说出中文
- 看到中文立即拼写出英文

**达标标准：** 正确率 ≥ 90%

**未达标？** 先回顾高中词汇，否则后续学习会很吃力！

**推荐资源：**
- 高中英语3500词汇表
- 百词斩、扇贝单词（选择"高考词汇"）

---

### 4.3 四六级衔接词汇

**策略：** 不需要全部重新背！

- **已掌握的** → 跳过
- **模糊的** → 快速复习
- **完全陌生的** → 重点学习

**时间分配：** 1-2周快速过一遍即可

---

### 4.4 考研新增词汇（重点）

**特点：**
- 约2000词
- 多为学术词汇（科技、经济、社会等主题）
- 真题高频出现

**学习方法：**
1. 按主题分类学习（参考 `high_frequency_words.md`）
2. 结合词根词缀（参考 `word_groups.md`）
3. 从真题中提取，优先级最高

**时间分配：** 3-6个月，每天20-30个新词

---

### 4.5 熟词僻义（易错点）

**定义：** 你认识的单词,但考研考的是你不认识的意思

**示例：**

| 单词 | 常见义 | 熟词僻义（考研常考） |
|------|--------|---------------------|
| **develop** | 发展 | 患（疾病）：develop a disease |
| **address** | 地址 | 解决；演讲：address the problem / address the audience |
| **discipline** | 纪律 | 学科：an academic discipline |
| **appreciate** | 欣赏 | 理解；增值：I appreciate your concern / The value appreciated |
| **novel** | 小说（n.） | 新颖的（adj.）：a novel idea |
| **school** | 学校 | 学派：the Vienna school of thought |
| **capital** | 首都 | 资本；大写的：raise capital / capital letter |
| **article** | 文章 | 物品；条款：articles of clothing / Article 5 |
| **firm** | 公司（n.） | 坚定的（adj.）：a firm belief |
| **fine** | 好的 | 罚款：pay a fine |

**学习策略：**
- 整理真题中的熟词僻义
- 重点记忆在真题中的用法
- 做题时警惕"一眼认识"的单词

## 5. 每日学习计划模板

### 5.1 基础版（适合复习前期，基础薄弱）

```

10:00-10:30  学习新词 15-20个
             ├─ 理解词根词缀
             ├─ 记录常见搭配
             └─ 朗读例句3遍

12:00-12:15  午间复习（看英文回忆中文）
             └─ 复习上午刚学的新词

             ├─ 复习2天前学的词（第2次复习）
             ├─ 复习4天前学的词（第3次复习）
             └─ 复习7天前学的词（第4次复习）

22:00-22:30  睡前巩固
             ├─ 再过一遍今天的新词
             ├─ 重点看错题本
```

## 6. 熟词僻义攻克策略

### 6.1 什么是熟词僻义？

**定义：** 你熟悉的单词，但在考研真题中考查的是不常见的意思。

**为什么重要？**
- 考研阅读的"陷阱"：以为认识，实际理解错误
- 真题高频考点
- 影响阅读理解准确率

---

### 6.2 高频熟词僻义总结

#### 动词类

| 单词 | 基本义 | 熟词僻义（考研常考） |
|------|--------|---------------------|
| **develop** | 发展 | 患（病）、产生（问题）：develop a habit / develop cancer |
| **address** | 地址（n.） | 解决（问题）、演讲：address the issue / address the conference |
| **school** | 学校（n.） | 训练、教育（v.）：be schooled in classical music |
| **weather** | 天气（n.） | 经受住、度过（v.）：weather the storm |
| **compromise** | 妥协 | 危及、损害：compromise one's health |
| **appreciate** | 欣赏、感激 | 理解、意识到；增值：appreciate the complexity / The value appreciated |
| **entertain** | 娱乐 | 考虑（想法）：entertain the idea |
| **celebrate** | 庆祝 | 赞美、颂扬：celebrate sb's achievement |
| **discipline** | 纪律（n.） | 学科；训练（v.）：an academic discipline / discipline oneself |
| **exercise** | 锻炼 | 行使（权力）：exercise authority |

#### 名词类

| 单词 | 基本义 | 熟词僻义 |
|------|--------|---------|
| **article** | 文章 | 物品、条款：articles of clothing / Article 3 of the Constitution |
| **novel** | 小说（n.） | 新颖的（adj.）：a novel approach |
| **capital** | 首都 | 资本、大写字母：raise capital / capital letter |
| **firm** | 公司（n.） | 坚定的（adj.）：a firm belief |
| **fine** | 好的（adj.） | 罚款（n.）：pay a fine |
| **issue** | 问题 | 发行、发布：issue a statement / the latest issue of the magazine |
| **panel** | 面板 | 专家组、讨论小组：a panel of experts |
| **note** | 笔记 | 注意、钞票：take note of / a ten-dollar note |
| **character** | 性格、角色 | 文字、符号：Chinese characters |
| **practice** | 练习 | 惯例、行医：medical practice / in practice |

#### 形容词类

| 单词 | 基本义 | 熟词僻义 |
|------|--------|---------|
| **original** | 原始的 | 新颖的、独创的：an original idea |
| **liberal** | 自由的 | 慷慨的、大量的：a liberal donation |
| **fair** | 公平的 | 相当多的、晴朗的：a fair amount of time / fair weather |
| **particular** | 特别的 | 挑剔的：be particular about food |
| **present** | 在场的 | 现在的、目前的：at the present time |
| **immediate** | 立即的 | 紧挨着的、直接的：the immediate vicinity |

---

### 6.3 学习方法

**步骤：**

1. **整理真题熟词僻义**
   - 做真题时，遇到"认识但不理解"的词 → 标记
   - 查字典，确认是熟词僻义
   - 整理到专门的笔记本

2. **对比记忆**
   - 列出"常见义 vs. 熟词僻义"
   - 例句对比：
     - `develop a plan`（发展计划）
     - `develop a disease`（患病）

3. **真题语境强化**
   - 回到真题原文，理解熟词僻义的用法
   - 朗读含有熟词僻义的句子3-5遍

4. **定期复习**
   - 每周专门复习一次熟词僻义
   - 使用 vocabulary_tool.py 测试

## 7. 常见误区与解决方案

### 误区1：从A背到Z

**问题：**
- Abandon, abandon, abandon...（前几页反复背）
- 效率低，挫败感强

**解决方案：**
- ✅ 按**主题**分类（科技、经济、社会等）
- ✅ 按**词根**分组（-ject, -mit, -cede等）
- ✅ 按**真题频率**排序（高频词优先）

---

### 误区2：只背单词,不复习

**问题：**
- 今天背50个，明天忘45个
- 始终在"背新词 → 遗忘 → 再背"的循环

**解决方案：**
- ✅ **复习量应该是新词量的2-3倍**
- ✅ 严格遵循艾宾浩斯曲线（1/2/4/7/15天复习）
- ✅ 使用 vocabulary_tool.py 自动提醒复习

---

### 误区3：忽视词根词缀

**问题：**
- 孤立背单词，效率低
- 遇到生词完全无法推测

**解决方案：**
- ✅ 掌握30个核心词根
- ✅ 掌握20个常见前缀
- ✅ 掌握15个常见后缀
- ✅ 看到新词先拆解词根词缀

---

### 误区4：脱离真题背单词

**问题：**
- 背了很多"生僻词"，真题不考
- 真题高频词没掌握

**解决方案：**
- ✅ **真题词汇优先级最高**
- ✅ 从真题阅读中提取生词
- ✅ 参考真题词频表

---

### 误区5：只看不写

**问题：**
- 看到单词认识，写作时想不起来
- 阅读能力提高,写作能力没提高

**解决方案：**
- ✅ 定期进行"看中文写英文"测试
- ✅ 用新学的单词造句
- ✅ 在作文中主动使用新词

---

### 误区6：急功近利

**问题：**
- 想在1个月内背完5500词
- 贪多嚼不烂，质量差

**解决方案：**
- ✅ **质量>数量**：宁可少背几个，也要记牢
- ✅ 设定合理目标：每天20-30个新词
- ✅ 长期坚持：3-6个月稳步提升

---

## 总结：词汇学习黄金法则

### 三大核心原则

1. **科学复习 > 盲目背新词**
   - 遵循艾宾浩斯曲线
   - 复习量 = 新词量 × 2-3

2. **理解记忆 > 机械记忆**
   - 词根词缀法
   - 语境记忆法

3. **真题导向 > 词汇书**
   - 真题高频词优先
   - 从阅读中学词汇
