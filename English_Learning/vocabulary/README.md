# 词汇学习子模块

## 模块简介

本子模块专注于**考研英语一词汇学习方法论**，而非简单提供词汇列表。通过科学的记忆方法、系统的词根词缀体系和主题化高频词汇分类，帮助考生高效掌握考研词汇。

---

## 核心理念

❌ **不是什么**：从A到Z的5500词汇表（效率低下！）

✅ **是什么**：
- 科学记忆方法（艾宾浩斯曲线、词根词缀法、语境记忆法）
- 词根词缀体系（30+核心词根，举一反三）
- 主题化高频词汇（8大主题，覆盖真题高频词）
- 智能学习工具（Python交互式学习、错题本、复习提醒）

---

## 文件说明

### 1. vocabulary_guide.md（词汇学习方法论）

**内容概览：**
- 艾宾浩斯遗忘曲线与复习计划
- 词根词缀记忆法详解
- 语境记忆法（搭配+例句）
- 分级学习路线（基础→衔接→考研新增）
- 每日学习计划模板
- 熟词僻义总结框架

**适合人群：** 所有考生，尤其是词汇基础薄弱或之前死记硬背效果不佳者

---

### 2. word_groups.md（词根词缀分类词汇组）

**内容概览：**
- 30+核心词根（如 -ject, -mit, -cede, -duct, -port 等）
- 每个词根衍生 5-10 个单词
- 常见前缀（un-, re-, pre-, dis-, anti- 等）
- 常见后缀（-tion, -ment, -ness, -ful, -able 等）

**使用方法：**
1. 先理解词根含义
2. 学习衍生词时联想词根
3. 通过词根记忆，一次记住一组词

**示例：**
```
词根：-ject（投掷）
├─ inject (v.) 注入（in-向内 + ject投掷）
├─ reject (v.) 拒绝（re-向后 + ject投掷 → 扔回去）
├─ project (v./n.) 投射；项目（pro-向前 + ject投掷）
├─ eject (v.) 弹出（e-向外 + ject投掷）
└─ subject (n./v.) 主题；使服从（sub-在下 + ject投掷）
```

---

### 3. high_frequency_words.md（高频词汇主题分类）

**内容概览：**
- 8大主题分类：科技、经济、社会、教育、法律、医学健康、环境、文化
- 每个主题 15-20 个核心词汇
- 包含中文释义和常见搭配

**优势：**
- 主题记忆更有上下文
- 便于阅读理解中快速识别主题
- 符合真题出题规律（真题文章多为主题性文章）

---

### 4. vocabulary_tool.py（词汇学习工具）

**功能特性：**

| 功能 | 说明 |
|------|------|
| 随机测试 | 中→英 / 英→中 双向测试 |
| 艾宾浩斯复习提醒 | 基于学习日期自动提醒复习（1/2/4/7/15天） |
| 错题本管理 | 自动保存错误单词，强化练习 |
| 学习进度统计 | 已学词汇数、正确率、连续学习天数 |
| 词汇搜索 | 快速查找单词释义 |
| 数据导入/导出 | JSON格式，方便备份和迁移 |

**使用方法：**

```bash
# 方法1：独立运行
python vocabulary_tool.py

# 方法2：通过主程序
python ../main.py --vocab

# 方法3：在代码中导入
from vocabulary.vocabulary_tool import VocabularyTool
tool = VocabularyTool()
tool.run()
```

**数据文件位置：**
- `user_data/words.json` - 词汇库
- `user_data/wrong_words.json` - 错题本
- `user_data/learning_log.json` - 学习记录

## 学习路线建议

### 第一阶段：建立方法论认知（1-2天）

1. 阅读 `vocabulary_guide.md`，理解科学记忆原理
2. 制定个人学习计划（每日新词量、复习时间）

### 第二阶段：掌握词根词缀体系（1周）

1. 学习 `word_groups.md` 中的30+核心词根
2. 每天学习 3-5 个词根及其衍生词
3. 使用 `vocabulary_tool.py` 进行测试

### 第三阶段：主题词汇扩展（2-3周）

1. 按照 `high_frequency_words.md` 的8大主题学习
2. 结合真题阅读文章，在语境中记忆
3. 每天使用工具复习，遵循艾宾浩斯曲线

### 第四阶段：真题词汇强化（持续）

1. 从真题中提取生词，添加到个人词库
2. 重点攻克熟词僻义
3. 每周回顾错题本，反复强化

## 学习技巧

### ✅ DO（推荐做法）

- **遵循艾宾浩斯曲线**：复习比学新词更重要
- **词根词缀优先**：理解词根，举一反三
- **多维记忆**：词根+语境+搭配，三管齐下
- **主题记忆**：按主题分类，建立知识网络
- **真题导向**：以真题词汇为核心，不贪多求全

### ❌ DON'T（避免陷阱）

- 不要从A背到Z（效率极低）
- 不要只背单词书不看真题
- 不要忽视复习（会前功尽弃）
- 不要孤立记忆（缺乏上下文）
- 不要贪多嚼不烂（质量>数量）

## 常见问题

**Q1：我需要背完5500词吗？**

A：不需要。考研真题高频词约2000个，加上基础词汇3500个，掌握这些足够。本模块重点提供这些词汇的学习方法和框架。

---

**Q2：词根词缀法对所有单词都有效吗？**

A：约70-80%的考研词汇可以通过词根词缀记忆。剩余20-30%（如简单词、外来词）需要通过语境和重复记忆。

---

**Q3：每天应该学多少新词？**

A：建议：
- 基础薄弱：15-20个/天
- 基础中等：20-30个/天
- 基础较好：30-50个/天

**关键：复习量应该是新词量的2-3倍！**

---

**Q4：vocabulary_tool.py 的词库从哪里来？**

A：工具内置了30个示例单词用于演示。实际使用时，建议：
1. 从真题中提取生词
2. 参考 `high_frequency_words.md` 手动添加
3. 使用工具的导入功能批量添加

---

**Q5：艾宾浩斯复习提醒如何使用？**

A：工具会自动记录每个单词的学习日期，并在第 1/2/4/7/15 天提醒复习。只需每天打开工具，按提示复习即可。
