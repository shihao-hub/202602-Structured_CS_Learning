## 1. 为什么要识别文章结构?

### 1.1 结构识别的重要性

**好处**:

1. **提高阅读速度**: 知道接下来要讲什么,预判内容
2. **把握文章重点**: 不同结构的重点不同
3. **提高主旨题正确率**: 结构决定主旨
4. **优化时间分配**: 知道哪些段落可以略读

---

### 1.2 考研阅读五大结构模式

| 结构类型 | 出题频率 | 阅读重点 |
|---------|---------|---------|
| **问题-解决型** | ⭐⭐⭐⭐⭐ | 问题是什么?解决方案是什么? |
| **现象-解释型** | ⭐⭐⭐⭐ | 现象是什么?原因/机制是什么? |
| **观点-论证型** | ⭐⭐⭐⭐⭐ | 作者观点是什么?论据是什么? |
| **对比-评价型** | ⭐⭐⭐ | A和B有什么不同?作者倾向哪个? |
| **叙事-评论型** | ⭐⭐ | 故事内容 + 作者评论/寓意 |

## 2. 问题-解决型(Problem-Solution)

### 2.1 结构特征

**模式**:

```
引出问题 → 分析问题(原因/影响) → 提出解决方案 → 评价方案/展望
```

**核心**: 文章围绕一个**问题**及其**解决方案**展开

---

### 2.2 信号词

**引出问题**:
- problem, issue, challenge, crisis
- concern, threat, difficulty

**分析问题**:
- cause, reason, result from
- impact, consequence, effect

**提出方案**:
- solution, approach, strategy
- address, tackle, solve, resolve
- measure, policy, initiative

**评价方案**:
- effective, successful, promising
- however, limitation, drawback

---

### 2.3 文章骨架示例

**主题**: 海洋塑料污染

```
Para 1: 引出问题
        海洋塑料污染日益严重,威胁海洋生态

Para 2: 分析问题(规模)
        每年800万吨塑料进入海洋

Para 3: 分析问题(影响)
        对海洋生物和人类健康的危害

Para 4: 解决方案1
        减少一次性塑料使用

Para 5: 解决方案2
        研发可降解塑料

Para 6: 解决方案3
        海洋塑料清理技术

Para 7: 总结/展望
        需要全球合作,综合施策
```

---

### 2.4 阅读策略

**重点关注**:
- ✅ 问题的性质和严重性
- ✅ 解决方案的内容
- ✅ 方案的优缺点

**可以略读**:
- 问题的详细数据(除非题目问到)
- 问题的具体案例

**主旨**:
- 通常是"如何解决XX问题"

---

### 2.5 真题示例分析

**2019年Text 1(节选)**:

```
Para 1: 引出问题 - 金融危机后,经济学家信任度下降
Para 2: 分析问题 - 经济学预测失败的原因
Para 3: 解决方案1 - 经济学需要更多跨学科研究
Para 4: 解决方案2 - 经济学家应该更谦虚
Para 5: 总结 - 改革经济学教育
```

**主旨**: 如何恢复经济学的可信度

## 3. 现象-解释型(Phenomenon-Explanation)

### 3.1 结构特征

**模式**:

```
描述现象 → 解释原因/机制 → 深入分析 → 结论/启示
```

**核心**: 文章先呈现一个**有趣/反常的现象**,然后**解释为什么**

---

### 3.2 信号词

**描述现象**:
- phenomenon, trend, pattern
- observe, notice, find
- surprising, unexpected, interesting

**解释原因**:
- reason, cause, explanation
- because, due to, result from
- mechanism, process

**得出结论**:
- therefore, thus, this suggests
- implication, significance

---

### 3.3 文章骨架示例

**主题**: 为什么成功人士早起?

```
Para 1: 描述现象
        许多成功人士都有早起习惯

Para 2: 列举例子
        CEO们的早起案例

Para 3: 解释原因1(生物学)
        早晨大脑皮层最活跃

Para 4: 解释原因2(心理学)
        早起给人控制感和成就感

Para 5: 解释原因3(社会学)
        早晨干扰少,效率高

Para 6: 结论/启示
        养成早起习惯的建议
```

---

### 3.4 阅读策略

**重点关注**:
- ✅ 现象是什么?
- ✅ 为什么会出现这个现象?(原因/机制)
- ✅ 这个现象说明了什么?(启示)

**可以略读**:
- 现象的详细描述(抓住核心即可)

**主旨**:
- 通常是"XX现象及其原因"

---

### 3.5 真题示例分析

**2018年Text 2(节选)**:

```
Para 1: 现象 - 美国人工作时间增加,休息时间减少
Para 2: 数据支持 - 统计数据显示趋势
Para 3: 原因1 - 技术进步模糊了工作与生活界限
Para 4: 原因2 - 经济压力增加
Para 5: 影响 - 对健康和幸福感的负面影响
Para 6: 启示 - 需要重新平衡工作与生活
```

**主旨**: 美国人休息时间减少的原因及影响

## 4. 观点-论证型(Argument-Support)

### 4.1 结构特征

**模式**:

```
提出观点/论点 → 论据1 → 论据2 → 论据3 → (反驳反方) → 重申论点
```

**核心**: 作者提出一个**观点**,然后用**论据**支持

---

### 4.2 信号词

**提出观点**:
- argue, claim, believe
- maintain, assert, contend
- the author's view is...

**提供论据**:
- evidence, proof, data
- research shows, studies indicate
- for example, for instance

**反驳反方**(可选):
- however, critics argue
- some people believe, but...
- while it is true that..., nevertheless...

**重申论点**:
- in conclusion, therefore
- this proves, this demonstrates

---

### 4.3 文章骨架示例

**主题**: 远程工作比办公室工作更高效

```
Para 1: 提出论点
        远程工作可以提高生产力

Para 2: 论据1(时间节省)
        无需通勤,节省每天2小时

Para 3: 论据2(灵活性)
        员工可以在最佳状态时工作

Para 4: 论据3(数据支持)
        研究显示远程工作者效率提高20%

Para 5: 反驳反方观点
        虽然有人担心沟通问题,但视频会议可以解决

Para 6: 重申论点+展望
        远程工作是未来趋势
```

---

### 4.4 阅读策略

**重点关注**:
- ✅ 作者的中心论点是什么?
- ✅ 用了哪些论据支持?(实验/数据/案例)
- ✅ 有没有反驳反方观点?
- ✅ 作者态度(支持/反对)

**可以略读**:
- 论据的详细过程(抓住结论即可)

**主旨**:
- 通常就是作者的中心论点

---

### 4.5 真题示例分析

**2020年Text 3(节选)**:

```
Para 1: 论点 - 人工智能不会完全取代人类工作
Para 2: 论据1 - AI缺乏创造力
Para 3: 论据2 - AI无法处理复杂社会情境
Para 4: 论据3 - 历史上技术进步创造了更多工作
Para 5: 反驳 - 虽然某些工作会消失,但会有新工作出现
Para 6: 结论 - 人类应该适应,而非恐惧
```

**主旨**: AI不会完全取代人类工作

## 5. 对比-评价型(Comparison-Evaluation)

### 5.1 结构特征

**模式**:

```
引出对比对象(A vs B) → A的特点 → B的特点 → 对比分析 → 作者评价
```

**核心**: 对比两个事物,分析优劣,**作者通常有倾向性**

---

### 5.2 信号词

**引出对比**:
- compare, contrast, difference
- A and B, A versus B

**对比分析**:
- while, whereas, on the other hand
- in contrast, by comparison
- similarly, likewise, both

**评价优劣**:
- superior, inferior, better, worse
- advantage, disadvantage
- more effective, less efficient

---

### 5.3 文章骨架示例

**主题**: 传统媒体 vs 社交媒体

```
Para 1: 引出对比
        传统媒体和社交媒体各有特点

Para 2: 传统媒体特点
        专业、权威、但传播慢

Para 3: 传统媒体案例
        报纸、电视的优势

Para 4: 社交媒体特点
        快速、互动、但可能不准确

Para 5: 社交媒体案例
        Twitter、Facebook的影响

Para 6: 对比分析
        两者在信息传播中的不同作用

Para 7: 作者评价
        两者应该互补,而非对立
```

---

### 5.4 阅读策略

**重点关注**:
- ✅ 对比的两个对象是什么?
- ✅ 它们有什么不同?(特点/优缺点)
- ✅ 作者倾向哪个?(或认为应该如何平衡)

**制作对比表**:

| 方面 | A | B |
|------|---|---|
| 特点1 | ... | ... |
| 特点2 | ... | ... |
| 优势 | ... | ... |
| 劣势 | ... | ... |

**主旨**:
- 通常是"A和B的对比及作者观点"

---

### 5.5 真题示例分析

**2017年Text 4(节选)**:

```
Para 1: 引出对比 - 公立学校 vs 私立学校
Para 2: 公立学校特点 - 资源有限,但包容性强
Para 3: 私立学校特点 - 资源丰富,但可能加剧不平等
Para 4: 对比数据 - 学业表现对比
Para 5: 作者评价 - 应该提升公立学校质量,而非鼓励私立
```

**主旨**: 公立学校vs私立学校的对比及教育公平

## 6. 叙事-评论型(Narrative-Comment)

### 6.1 结构特征

**模式**:

```
叙述故事/事件 → 作者评论 → 引申意义 → 总结启示
```

**核心**: 通过**讲故事**来说明**道理**

---

### 6.2 信号词

**叙述故事**:
- story, case, event, incident
- once, in the past, recently
- describe, recount

**作者评论**:
- this illustrates, this shows
- the significance of...
- what this means is...

**引申意义**:
- broader implication
- reflect, reveal, demonstrate
- this is not just about...

---

### 6.3 文章骨架示例

**主题**: 一个失败的创业故事及启示

```
Para 1: 引入故事
        介绍主人公和背景

Para 2-3: 叙述故事
          创业过程、遇到的困难、最终失败

Para 4: 作者评论1
        失败的原因分析

Para 5: 作者评论2
        这个故事反映的普遍问题

Para 6: 引申意义
        对创业者的启示

Para 7: 总结
        失败也是宝贵经验
```

---

### 6.4 阅读策略

**重点关注**:
- ✅ 故事的核心情节(不要陷入细节)
- ✅ 作者的评论(这才是重点!)
- ✅ 故事要说明什么道理?(引申意义)

**注意**:
- 故事本身不重要,重要的是故事的**寓意**
- 类似"例证题"的逻辑

**主旨**:
- 通常是故事要说明的道理

---

### 6.5 真题示例分析

**2016年Text 1(节选)**:

```
Para 1-2: 叙述 - 法国经济学家托马斯·皮凯蒂的故事
Para 3: 评论 - 他的研究挑战了传统经济学观点
Para 4: 引申 - 这反映了经济不平等问题的严重性
Para 5: 启示 - 社会需要重新审视财富分配
```

**主旨**: 通过经济学家的故事,说明经济不平等问题

## 7. 实战应用

### 7.1 快速识别文章结构(30秒)

**步骤**:

1. **看首段**: 判断是引出问题?描述现象?提出观点?
2. **扫各段首句**: 是论据?解决方案?对比?
3. **看尾段**: 是总结?评价?启示?

**快速判断**:

| 首段特征 | 可能的结构 |
|---------|-----------|
| 提出问题/挑战 | 问题-解决型 |
| 描述有趣/反常现象 | 现象-解释型 |
| 明确提出观点 | 观点-论证型 |
| 引入两个对比对象 | 对比-评价型 |
| 开始讲故事 | 叙事-评论型 |

---

### 7.2 根据结构调整阅读策略

**问题-解决型**:
- 重点: 问题+解决方案
- 略读: 问题的详细描述

**现象-解释型**:
- 重点: 现象+原因/机制
- 略读: 现象的案例

**观点-论证型**:
- 重点: 论点+论据+作者态度
- 略读: 论据的详细过程

**对比-评价型**:
- 重点: 对比差异+作者倾向
- 略读: 某一方的详细描述

**叙事-评论型**:
- 重点: 作者评论+引申意义
- 略读: 故事的细节

---

### 7.3 结构与主旨题

**不同结构的主旨通常是**:

| 结构类型 | 主旨通常是 |
|---------|-----------|
| 问题-解决型 | 如何解决XX问题 |
| 现象-解释型 | XX现象及其原因 |
| 观点-论证型 | 作者的中心论点 |
| 对比-评价型 | A vs B的对比及作者观点 |
| 叙事-评论型 | 故事要说明的道理 |

---

### 7.4 练习建议

**方法**:

1. **做真题时,标注文章结构**
   - 在每段旁边写: 引入/论据1/转折/总结...

2. **总结结构规律**
   - 某类主题(如科技、教育)常用哪种结构?

3. **预判内容**
   - 看到首段,预测接下来会讲什么
   - 验证预测是否正确

| 结构 | 信号词 | 阅读重点 | 主旨特征 |
|------|-------|---------|---------|
| **问题-解决** | problem, solution, address | 问题+方案 | 如何解决XX |

| **叙事-评论** | story, illustrate, significance | 评论+寓意 | 故事的道理 |
